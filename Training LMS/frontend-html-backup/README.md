## 🔧 配置说明

### 模拟数据模式（当前模式）

默认使用模拟数据，无需任何配置，开箱即用！

所有功能都可以正常展示和交互，但数据为前端模拟，不会真实保存。

### 真实数据模式

如需连接真实的Supabase后端：

1. 打开 `app.js` 文件
2. 修改第6-7行的Supabase配置：

```javascript
const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_ANON_KEY = 'your-real-anon-key';
```

3. 刷新页面即可自动切换到真实API模式

## 🚀 页面路由

系统采用单页面应用（SPA）设计，无需刷新即可切换视图：

| 路由 | 页面名称 | 可见角色 |
|------|---------|---------|
| `dashboard` | 数据看板 | 所有角色 |
| `my-courses` | 我的课程 | 学员 |
| `courses` | 课程管理 | 培训负责人及以上 |
| `exams` | 考试管理 | 培训负责人及以上 |
| `students` | 学员管理 | 门店管理员及以上 |
| `stores` | 门店管理 | 超级管理员/管理员 |
| `stats` | 统计分析 | 所有角色 |

