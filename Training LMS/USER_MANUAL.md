## 系统简介

SmartIce培训管理系统是专为餐饮运营企业设计的员工培训平台，帮助员工快速掌握岗位技能和企业价值观。

### 核心功能

- **在线学习**: 随时随地学习课程内容
- **在线考试**: 自动判分，即时查看成绩
- **学习追踪**: 记录学习进度，数据可视化
- **智能补考**: 自动计算补考时间，最多3次机会
- **价值观培训**: 融入企业文化，价值观题目占比30%

### 用户角色

- **L1-L3员工**: 学习课程、参加考试
- **L4-L5+管理层**: 员工端功能 + 管理后台权限

## 员工端使用指南

### 1. 登录系统

1. 访问系统地址：`http://your-domain.com`
2. 输入用户名和密码
3. 点击"登录"按钮

**首次登录**：
- 用户名和密码由管理员创建后提供
- 建议首次登录后修改密码

### 2. 浏览课程

登录后默认进入"课程中心"页面，您将看到：

- **课程列表**: 显示所有可学习的课程
- **课程分类**: 前厅、厨房、管理、价值观
- **学习进度**: 显示每门课程的学习进度百分比
- **课程状态**: 未开始、学习中、已完成

**操作步骤**：
1. 浏览课程列表
2. 点击课程卡片查看详情
3. 点击"开始学习"进入课程

### 3. 学习课程

#### 查看课程详情

进入课程详情页后，您可以看到：
- 课程简介和学习目标
- 章节列表
- 关联考试信息
- 学习进度统计

#### 学习章节内容

1. 点击任一章节进入学习页面
2. 阅读文档内容、观看视频（如有）
3. 完成后点击"标记为已完成"
4. 自动跳转到下一章节

**学习提示**：
- 系统会自动保存学习进度
- 可随时暂停，下次继续
- 建议按顺序学习，循序渐进

### 4. 参加考试

#### 开始考试

1. 在课程详情页找到"关联考试"
2. 确认满足参考条件（如需完成课程）
3. 点击"开始考试"

**考试规则**：
- 每场考试限时（通常60-90分钟）
- 题型包括：单选、多选、判断题
- 需达到及格分数（通常60分）才能通过
- 最多允许3次考试机会

#### 答题

1. 仔细阅读每道题目
2. 选择正确答案
3. 可随时修改已答题目
4. 点击"下一题"继续
5. 完成所有题目后点击"提交试卷"

**答题提示**：
- 单选题：只能选一个答案
- 多选题：可选择多个答案，必须全对才得分
- 判断题：选择正确或错误
- 注意时间管理，避免超时

#### 查看成绩

提交试卷后立即显示成绩：
- 总分和得分
- 是否通过
- 正确答案解析
- 错误题目回顾

**未通过怎么办？**
- 系统会自动计算下次补考时间（通常间隔3天）
- 最多允许2次补考机会
- 建议复习错题内容后再考

### 5. 学习数据看板

点击顶部导航栏"学习数据"查看个人学习情况：

- **学习统计**: 已学课程数、学习时长、完成进度
- **考试记录**: 参加考试次数、通过率、平均分
- **学习轨迹**: 最近学习的课程和章节
- **成长曲线**: 学习进度时间轴

## 管理后台使用指南

### 1. 访问管理后台

**权限要求**: 仅限L4及以上管理层

**访问方式**:
1. 登录员工端
2. 点击右上角头像
3. 选择"管理后台"（如有权限）
4. 或直接访问：`http://your-domain.com/admin`

### 2. 数据看板

管理后台首页显示系统整体数据：

- **用户统计**: 总用户数、活跃用户、新增用户
- **课程统计**: 课程总数、学习人次、完成率
- **考试统计**: 考试总数、参考人次、通过率
- **题库统计**: 题目总数、题型分布、难度分布

**快捷操作**:
- 点击统计卡片可跳转到对应管理页面
- 支持按时间筛选数据（今日、本周、本月）

### 3. 用户管理

#### 查看用户列表

- **搜索**: 支持按用户名、邮箱、部门搜索
- **筛选**: 按角色、部门、状态筛选
- **排序**: 按注册时间、最后登录排序

#### 管理用户

**启用/禁用用户**:
1. 找到目标用户
2. 点击"禁用"或"启用"按钮
3. 确认操作

**重置密码**:
1. 点击用户行的"重置密码"按钮
2. 系统自动生成随机密码
3. 复制密码并安全传达给用户

**添加用户**:
1. 点击"新增用户"按钮
2. 填写用户信息（用户名、邮箱、角色、部门）
3. 设置初始密码
4. 点击"创建"

### 4. 课程管理

#### 浏览课程

- 查看所有课程列表
- 按分类筛选（前厅/厨房/管理/价值观）
- 查看课程状态（已发布/草稿）

#### 编辑课程

1. 点击课程的"编辑"按钮
2. 修改课程信息：
   - 课程名称和简介
   - 封面图片
   - 学习目标
   - 关联考试
3. 点击"保存"

#### 管理章节

1. 在课程编辑页点击"章节管理"
2. 添加新章节：
   - 章节标题
   - 章节简介
   - 上传学习内容（文档/视频）
3. 调整章节顺序（拖拽排序）
4. 删除章节（谨慎操作）

#### 发布课程

1. 确认课程内容完整
2. 点击"发布"按钮
3. 课程将对所有员工可见

### 5. 题库管理

#### 浏览题目

- 查看所有题目列表（当前556道）
- 按题型筛选（单选/多选/判断/简答）
- 按难度筛选（简单/中等/困难）
- 按类别筛选（技能/价值观）

#### 添加题目

1. 点击"新增题目"按钮
2. 填写题目信息：
   - 题目内容
   - 题型选择
   - 难度等级
   - 类别（技能或价值观类型）
3. 设置选项和正确答案
4. 添加答案解析（可选）
5. 点击"创建"

#### 编辑/删除题目

- **编辑**: 点击题目行的"编辑"按钮
- **删除**: 点击"删除"按钮（⚠️ 已用于考试的题目谨慎删除）

#### 批量导入题目

1. 点击"批量导入"按钮
2. 下载Excel模板
3. 按模板格式填写题目
4. 上传Excel文件
5. 系统自动导入并验证

### 6. 考试管理

#### 创建考试

1. 点击"新增考试"按钮
2. 填写考试基本信息：
   - 考试名称
   - 关联课程
   - 考试时长（分钟）
   - 及格分数
   - 最大尝试次数
   - 补考冷却期（天）
3. 选择题目：
   - 手动选择题目
   - 或按条件随机抽题
4. 预览试卷
5. 点击"创建"

#### 管理考试

- **启用/禁用**: 控制考试是否对学员可见
- **编辑**: 修改考试配置（⚠️ 已有考试记录时谨慎修改）
- **查看成绩**: 查看所有参考学员成绩

#### 导出成绩

1. 进入考试详情页
2. 点击"导出成绩"按钮
3. 选择导出格式（Excel/CSV）
4. 下载文件

**导出内容包括**:
- 考生姓名、工号
- 考试分数
- 是否通过
- 考试时间
- 答题详情

### 7. 权限说明

## 常见问题

### 1. 登录问题

**Q: 忘记密码怎么办？**

A: 联系您的直属主管或管理员重置密码。系统会生成临时密码，请及时修改。

**Q: 提示"用户已被禁用"？**

A: 账号可能被临时禁用，请联系管理员了解原因并申请恢复。

### 2. 学习问题

**Q: 无法看到某些课程？**

A: 可能原因：
- 课程尚未发布
- 您的角色/部门不在授权范围
- 需要先完成前置课程

**Q: 视频无法播放？**

A: 检查：
- 网络连接是否正常
- 浏览器是否支持（推荐Chrome/Edge）
- 尝试刷新页面

**Q: 学习进度没有保存？**

A: 确保：
- 点击了"标记为已完成"按钮
- 网络连接稳定
- 未使用浏览器的无痕模式

### 3. 考试问题

**Q: 考试中途断网怎么办？**

A:
- 系统会自动保存已答题目
- 重新连接后可继续考试
- 如超时未提交，视为本次考试作废

**Q: 第一次没通过，什么时候能补考？**

A:
- 系统会自动计算补考时间（通常3天后）
- 在"我的考试"页面查看下次可考时间
- 建议利用这段时间复习错题

**Q: 为什么不能立即补考？**

A:
- 补考冷却期帮助您充分复习
- 防止盲目刷题，确保真正掌握知识
- 符合培训管理最佳实践

**Q: 3次机会都用完了怎么办？**

A:
- 联系培训主管申请特批
- 可能需要参加线下培训后再考
- 系统管理员可重置考试机会

### 4. 技术问题

**Q: 推荐使用什么浏览器？**

A: 推荐使用以下浏览器最新版本：
- ✅ Google Chrome
- ✅ Microsoft Edge
- ✅ Firefox
- ⚠️ Safari（部分功能可能不兼容）
- ❌ IE浏览器（不支持）

**Q: 手机可以使用吗？**

A:
- ✅ 支持手机浏览器访问
- ✅ 支持学习课程内容
- ✅ 支持参加考试
- ⚠️ 管理后台建议使用电脑

**Q: 数据会丢失吗？**

A:
- 系统每天自动备份数据
- 您的学习记录永久保存
- 服务器位于内网，数据安全可控

### 5. 其他问题

**Q: 如何给系统提建议？**

A:
- 联系管理员反馈
- 发送邮件至：training@smartice.com
- 在用户满意度调查中提出

**Q: 系统有手机App吗？**

A:
- 目前暂无独立App
- 可使用手机浏览器访问
- 计划未来开发小程序版本

**Q: 可以下载课程内容吗？**

A:
- 出于版权保护，暂不支持下载
- 支持在线学习，随时随地访问
- 部分课件可向管理员申请
