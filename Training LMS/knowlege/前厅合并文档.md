## 一、通用管理标准

### 通用服务礼仪

#### 客人问候标准
- [ ] **所有岗位**遇到客人时,必须主动问候
- [ ] 问候语:"您好"
- [ ] 问候时保持微笑,眼神友善
- [ ] 问候时可配合点头手势
- [ ] 工作中遇到客人,应暂停工作或避让,优先让客人通行
- [ ] 不得无视客人,不得只顾工作不打招呼

#### 服务态度标准
- [ ] 保持热情、友善、耐心的服务态度
- [ ] 面带微笑,语气温和
- [ ] 主动服务,及时响应客人需求
- [ ] 尊重客人,不与客人争执
- [ ] 遇到投诉,先道歉后处理

### 1.1 仪容仪表标准

#### 男士标准
- [ ] 发型:头发前不过眉、侧不过耳、后不过领,保持短发整洁
- [ ] 面容:干净整洁,**严禁留长胡须**,佩戴口罩
- [ ] 饰品:严禁佩戴夸张饰品,可佩戴婚戒/手表
- [ ] 衣服/工牌:标准工服穿戴齐备、干净整洁、无异味、无皱褶;工牌佩戴规范
- [ ] 手部:指甲不超过指尖2mm,干净整洁无污渍;指甲内无污垢
- [ ] 裤子:统一按照工装要求,干净无皱褶、无异味
- [ ] 鞋袜:黑色鞋,禁止拖鞋/凉鞋;袜子黑色或深色
- [ ] 站姿:挺胸抬头,禁止驼背、吊儿郎当

#### 女士标准
- [ ] 发型:梳理整洁,先梳成马尾、再用发卷捆住或盘起;刘海不遮眉毛、无耳发
- [ ] 面容:干净整洁,可化淡妆、不允许浓妆,佩戴口罩
- [ ] 饰品:严禁佩戴夸张饰品,可佩戴婚戒/手表
- [ ] 衣服/工牌:标准工服穿戴齐备、干净整洁、无异味、无皱褶;工牌佩戴规范
- [ ] 手部:指甲不超过指尖2mm,干净整洁无污渍;**不能做美甲**(涉及食品安全风险)
- [ ] 裤子:统一按照工装要求,干净无皱褶、无异味
- [ ] 鞋袜:黑色鞋,禁止拖鞋/凉鞋;袜子黑色或深色
- [ ] 站姿:挺胸抬头,禁止驼背、吊儿郎当

### 1.2 毛巾使用标准

#### 毛巾分类使用
> **说明**:毛巾按照使用场景进行颜色区分,避免交叉污染

- [ ] **蓝色毛巾**:桌面清洁(一次擦拭)
- [ ] **紫色毛巾**:桌面抛光(二次擦拭)
- [ ] **白色毛巾**:餐具擦拭
- [ ] **棕色毛巾**:洗手间清理
- [ ] **绿色毛巾**:椅凳清理

#### 清洗及消毒
- [ ] 洗洁精加热水清洗
- [ ] 84消毒液和热水比例1:9(1毫升消毒液9毫升水浸泡)
- [ ] 消毒浸泡30分钟

#### 使用要求
- [ ] 每日使用的毛巾需在14:00~14:30以及22:00~22:30期间更换或清洁一次(保证卫生标准)
- [ ] 定位摆放,不可交叉颜色使用(避免交叉污染)
- [ ] 毛巾禁止裁剪,每20天对不符合要求的毛巾淘汰更换(破损、严重污渍等情况)
- [ ] 前厅毛巾统一存放,分色分类管理(便于取用和管理)
- [ ] 每日晚市将所有毛巾消毒浸泡30分钟,清洗后统一晾晒在指定位置(确保第二天使用卫生)

### 1.3 托盘使用标准

#### 装盘原则
- [ ] 里重外轻:重物品放托盘内侧,轻物品放外侧
- [ ] 里后外先:先上的物品放外侧,后上的放内侧
- [ ] 里高外低:高的物品放内侧,低的放外侧

#### 行走步伐
- [ ] 保持托盘平稳,安全第一
- [ ] 汤水较多菜品:小步防止洒溅
- [ ] 狭窄通道或上楼梯:注意安全,稳步前行

### 1.4 进销存管理

#### 盘点标准
- [ ] **盘点时间**:每月月底最后一天;新店期(开业前2个月要求周盘点)
- [ ] **盘点范围**:库房、厅面、吧台、收银台
- [ ] **盘点的物品**:低值易耗、固定资产、物料、餐具、酒水(饮品)、客用增值物品
- [ ] **盘点标准**:以财务每月系统导出盘点表为标准
- [ ] **盘点的方法**:按固定路线分区盘点,从上方货架、中间台面、下方冰箱顺序逐一盘点

#### 先进先出规范
- [ ] 生产日期在前的优先出库;入库时间长的优先出库
- [ ] 所有入库物料相关标签齐备;批号/日期填写清楚规范准确
- [ ] 先到的物料先使用,后到的物料后使用
- [ ] 物料存放遵照先里后外、先下后上;取用时按相同顺序
- [ ] 成品、半成品、开封后的原料,加膜加盖存放于指定位置,**使用标签打印机打印效期标签**(而非手写),标签内容包括:品名、开封日期、效期日期、操作人
- [ ] 定期检查库存/效期,确保先进先出原则正确执行,确保库存中没有过期或临近过期产品

### 1.5 4D现场管理标准

#### 基本原则
- [ ] 区域内保持整洁卫生、工器具、原料等定位摆放,整齐有序
- [ ] 预制的半成品,应避免污染,与原料分开分类存放
- [ ] 食品按类别、品种分架,离地离墙(10厘米以上)整齐摆放
- [ ] 散装食品及原料储存容器加盖密封,同时经常检查,防止霉变
- [ ] 需冷藏的食材,需加膜加盖放置于指定冰箱
- [ ] 加工器具使用后及时清洗消毒,做到无霉、无锈、无渍、无残渣、定位摆放
- [ ] 操作时注意保持卫生,操作后及时清扫,保证地面、工作台面无异味、无残渣、无卫生死角
- [ ] 操作中产生的废弃物应边产生边处理,及时清理到有盖的容器内,做到一餐一清
- [ ] 完善防虫、防蝇、防鼠、防蟑螂、防尘设施,做到无虫、无蝇、无鼠、无蟑螂

#### 卫生检查标准

**地面卫生标准**
- [ ] 地面无水渍、油渍、污渍
- [ ] 地面无杂物、垃圾、食物残渣
- [ ] 地面干燥,无积水
- [ ] 地砖缝隙无污垢
- [ ] 地脚线无灰尘、蜘蛛网

**墙面卫生标准**
- [ ] 墙面无污渍、油渍、水渍
- [ ] 墙面无灰尘、蜘蛛网
- [ ] 瓷砖缝隙无污垢、发霉
- [ ] 墙面装饰物(画框、指示牌等)无灰尘

**台面卫生标准**
- [ ] 台面无水渍、油渍、污渍
- [ ] 台面无杂物、食物残渣
- [ ] 台面物品摆放整齐,定位管理
- [ ] 台面边缘、角落无污垢

**设备设施卫生标准**
- [ ] 设备表面无灰尘、油渍、污渍
- [ ] 设备缝隙、角落无污垢
- [ ] 设备内部定期清洁(参照设备清洁计划)
- [ ] 设备底部无灰尘、杂物
- [ ] 设备排风口、进风口无灰尘堵塞

**餐具卫生标准**
- [ ] 餐具清洗消毒合格,无水渍、油渍、污渍
- [ ] 餐具无破损、缺口
- [ ] 餐具存放整齐,加盖防尘
- [ ] 餐具擦拭使用白色毛巾，符合毛巾分类使用标准

**洗手间卫生标准**
- [ ] 洗手间无异味
- [ ] 地面干燥,无积水、污渍
- [ ] 洗手台无水渍、污渍
- [ ] 镜面无水渍、污渍
- [ ] 洗手间无污渍、异味
- [ ] 洗手间用品充足(纸巾、洗手液、除味剂)
- [ ] 垃圾桶及时清理,无异味

**卫生死角检查**
- [ ] 设备底部、后面
- [ ] 桌椅底部
- [ ] 墙角、地角
- [ ] 货架底部、背面
- [ ] 管道周围
- [ ] 下水道口

### 1.6 消防安全管理

#### 灭火器使用标准
- [ ] 保存在干燥通风且方便取用的地方,避免受潮、爆晒
- [ ] 定期检查各个部件是否完整,是否影响正常使用
- [ ] 检查是否达到规定的维修年限
- [ ] 对灭火器定期保养/维护/清洁
- [ ] 周围保持干净,禁止堆放杂物

#### 灭火器检查要点
- [ ] 检查压力表外表面无变形、损伤,指针处在绿区
- [ ] 喷嘴无变形、开裂及损伤
- [ ] 喷射软管畅通,无变形和损伤
- [ ] 压把、阀体等金属件无严重损伤、变形、锈蚀
- [ ] 保险销和铅封完好,未被开启喷射
- [ ] 筒体严重变形和锈蚀必须报废

#### 消防日常管理
- [ ] 安全出口通道不堆放任何物品
- [ ] 消防喷淋脱落不要用封箱带粘或胶水固定
- [ ] 定时查看各区域灭火器保质期,快过期及时上报更换
- [ ] 灭火器箱损坏无法维修要上报更换
- [ ] 应急电源及安全指示牌每周放一次电,检查设备运转是否正常
- [ ] 定期检查消火栓是否完好,有无生锈现象
- [ ] 检查消火栓供水闸阀是否渗漏水
- [ ] 消火栓定期放水检查
- [ ] 检查报警按钮、指示灯及控制线路

### 1.7 客诉处理标准

#### 客诉处理原则
- [ ] 先处理情绪,再处理问题
- [ ] 先道歉,再了解情况
- [ ] 迅速反应,及时处理
- [ ] 记录完整,跟进回访

#### 客诉分级标准

**第一类客诉**(员工/主管可处理):
- [ ] 菜品口味偏差(不是变质或异物)
- [ ] 上菜速度慢
- [ ] 服务态度一般性问题
- [ ] 餐具不够干净(无食品安全问题)
- [ ] 环境卫生小问题

**第二类客诉**(主管必须处理):
- [ ] 菜品中有异物
- [ ] 菜品变质
- [ ] 服务态度恶劣
- [ ] 价格争议
- [ ] 预定失误

**第三类客诉**(店长/区域经理/总公司处理):
- [ ] 食品安全事故(食物中毒)
- [ ] 人身伤害事故
- [ ] 财物丢失/损坏
- [ ] 媒体曝光
- [ ] 恶意投诉勒索

### 1.8 口碑管理标准

#### 线上评价管理
- [ ] 每日查看大众点评、美团等平台评价
- [ ] 24小时内回复所有评价(好评感谢,差评道歉并说明)
- [ ] 对差评进行电话回访,了解详情并改进
- [ ] 每周统计评分和差评原因,制定改进措施

#### 现场口碑促进
- [ ] 餐中体验回访:用餐过程中主动询问满意度
- [ ] 买单时引导:介绍线上活动,引导好评
- [ ] 会员维护:建立常客档案,提供个性化服务
- [ ] 及时处理投诉:当场解决问题,避免差评

### 1.9 垃圾管理标准

#### 垃圾分类
- [ ] **厨余垃圾**:剩菜剩饭等食物残渣
- [ ] **其他垃圾**:纸巾等一般垃圾
- [ ] **可回收物**:酒瓶、易拉罐等

#### 垃圾处理流程
- [ ] 垃圾桶2/3满时及时更换
- [ ] 一餐一清:每餐结束后集中清理
- [ ] 做好基本分类,避免完全混放即可
- [ ] 垃圾桶保持干净,无异味
- [ ] 晚市收市统一清运至指定区域

### 1.10 人员培训标准

#### 四步培训法
- [ ] **第一步:我讲你听** - 讲解工作流程和标准,确保员工理解操作要点
- [ ] **第二步:你讲我听** - 验收学习成果,检验员工是否理解到位
- [ ] **第三步:我做你看** - 示范操作流程,展示标准动作和细节
- [ ] **第四步:你做我看** - 检验实际操作,确认员工能够独立完成

#### 培训关键点
- [ ] 有计划地进行人员带训
- [ ] 重点跟踪带训人员进展情况
- [ ] 做到及时沟通,有问题及时解决
- [ ] 不能只提问题不给解决办法
- [ ] 不要轻易训斥店员,需耐心引导
- [ ] 每天关注人员训练和工作成果
- [ ] 及时发现问题并纠正引导
- [ ] 特别关注人员状态以及一些不规范的习惯

### 1.11 能源与设备管理标准

#### 能源检查标准
- [ ] **电力检查**:查看电费余额及电表情况,确保营业期间供电正常
- [ ] **燃气检查**:查看燃气余额及燃气表余量,确保正常供应
- [ ] **设备检查**:确认重要设备是否正常(电磁炉、空调、音响、抽排等)
- [ ] 检查是否有安全隐患
- [ ] 设备出现故障立即上报,及时解决;短时间内不能解决的,采用应急方案并向店长报备

#### 三关一闭标准
- [ ] **水关闭**:关闭所有水龙头,检查是否漏水
- [ ] **电关闭**:关闭所有不需要的电源(保留冰箱等必要设备)
- [ ] **燃气关闭**:关闭燃气总阀,确保无燃气泄漏
- [ ] **门闭锁**:检查所有门窗是否锁好,确保门店安全

#### 设备日常检查
- [ ] 检查电路、网络、电源插座是否正常无安全隐患,发现问题现场处理
- [ ] 检查设备是否正常运转(参照设备卡),异常设备立即维修或启用备用方案
- [ ] 检查冰箱内食品是否生熟分离,温度是否正常
- [ ] 检查前一天剩余食材是否变质变色变味
- [ ] 原料效期检查,是否变质/异味

### 1.12 标准流程模板

> **说明**: 为避免重复,以下模板适用于所有岗位。各岗位章节只列出特定要求,通用部分参照此模板执行。

#### 开市流程通用标准

**仪容仪表检查**
- [ ] 参照仪容仪表标准
- [ ] 工服整洁,工牌佩戴规范
- [ ] 对讲机测试(已于前一日收市时充电完成)

**班前会**
- [ ] 集合点名,清楚部门人员当日配置情况、到岗情况
- [ ] 检查人员仪容仪表,不符合标准的及时纠正
- [ ] 主管或店长传达工作安排及任务要求,做到:
  * 工作分配明确合理,指令清晰具体
  * 强调当日经营指标(营业额目标、好评率、收藏打卡率等)
  * 明确平台数据运营要求(大众点评、抖音等平台数据维护)
  * 明确会员数据运营任务(新增会员、会员复购等)
- [ ] 了解当日预定信息、估清菜品、活动信息
- [ ] 前一日问题复盘及改进措施提醒
- [ ] 当日重点工作及经营指标强调
- [ ] 协助配合其他岗位的餐前准备工作

**班前会时间**: 晚市开市前30分钟举行(上午时间较紧张,一般不单独组织班前会)

**开市环境及安全检查**
- [ ] 检查收市卫生工作是否按标准要求完成,不符合的现场立即整改,重大问题拍照上报
- [ ] 检查电路、网络、电源插座是否正常无安全隐患,发现问题现场处理
- [ ] 检查设备是否正常运转(参照设备卡),异常设备立即维修或启用备用方案
- [ ] 检查冰箱内食品是否生熟分离,温度是否正常,不符合标准立即调整
- [ ] 检查前一天剩余食材是否变质变色变味,问题食材立即封存报损
- [ ] 原料效期检查,是否变质/异味,临期食材优先使用,过期食材立即报损
- [ ] 检查原料/辅料是否充足,不足立即补充或调整当日估清
- [ ] 所有检查问题必须在开市前完成整改,无法整改的问题向店长报备应急方案
- [ ] 设备出现故障立即上报,及时解决;短时间内不能解决的,采用应急方案并向店总报备

**清洁卫生标准**
- [ ] 区域环境卫生(地面、墙面、操作台面)
- [ ] 设施设备卫生(参照设施设备卫生标准)
- [ ] 日常物品卫生

#### 营业中服务流程通用标准

- [ ] 清楚管理组负责人
- [ ] 对讲机开启并正常使用
- [ ] 严格按照产品SOP标准执行
- [ ] 查看食材/物料数量是否满足营业用量,不足时及时预制
- [ ] 查看辅料是否充足,根据当天预估量进行补充
- [ ] 废弃物应及时清理到有盖的废弃物容器内,废弃物容器标识明确、保持外观清洁、摆放合理,一餐一清
- [ ] 随时保持区域卫生干净

#### 收市流程通用标准

**原料收存与效期管理**
- [ ] 将没用完的原料收存于冰箱内,加膜加盖
- [ ] 使用标签打印机打印效期标签(日期、品名、责任人)
- [ ] 冰箱内物品分类、分区、分层、分颜色摆放整齐
- [ ] 检查存放的原料有无变质现象
- [ ] 对于变质不能用的原料,封存拍照并上报主管,记录报损台账

**清洁用具与物料管理**
- [ ] 用具、清洁用具清洗/消毒,按4D标准摆放
- [ ] 日常物料储存收纳,物品归位
- [ ] 对讲机、取号机等设备充电(确保次日开市前充满电)

**环境卫生清理**
- [ ] 区域环境卫生(地面、墙面、操作台面)
- [ ] 设施设备卫生清洁与维护
- [ ] 日常物料卫生清洁

**设备与安全检查**
- [ ] 设备检查(参照设备卡),记录设备运行状态
- [ ] 垃圾处理:集中放置在统一区域,生活垃圾/餐厨垃圾分开放置
- [ ] 电路、电源检查,关闭不必要设备
- [ ] 四害检查(蚊虫、老鼠、蟑螂等)

**三关一闭确认**
- [ ] 水关:关闭所有水龙头,检查是否漏水
- [ ] 电关:关闭所有不需要的电源(保留冰箱等必要设备)
- [ ] 燃气关:关闭燃气总阀,确保无燃气泄漏
- [ ] 门闭锁:检查所有门窗是否锁好,确保门店安全

## 二、管理岗位

### 2.1 店长工作流程

#### 岗位职责

负责管理门店所有日常运营工作和人员团队,包括工作分配和协调,人员培训及团队建设,对部分经营指标、顾客满意度以及团队人员满意度负责。

#### 开店流程

**1) 班前会与人员管理**

*标准流程*
- [ ] 组织开展班前会并对重点问题进行复盘总结(客诉、服务、工作流程等)
- [ ] 集合点名,清点当日人员情况
- [ ] 检查人员仪容仪表,不符合标准的及时纠正
- [ ] 根据集合传达的工作安排及任务要求做好工作拆分
- [ ] 分配各岗位人员进行开店工作
- [ ] 提醒各岗位工作重点及经营指标
- [ ] 了解当日预定信息、估清菜品、活动信息
- [ ] 对前一日出现的问题进行提醒并给出应对办法
- [ ] 针对性安排人员培训并验收成果

*关键点*
- [ ] 重点针对工作流程中出现的问题进行提醒与纠正
- [ ] 不窜岗,不拖延工作,及时沟通
- [ ] 重点强调经营指标工作:新增评价/收藏打卡数量、引导扫码团购、买单以及私域转化工作
- [ ] 重点跟踪带训人员进展情况,做到及时沟通,有问题及时解决
- [ ] 不能只提问题不给解决办法
- [ ] 不要轻易训斥店员,需耐心引导

*培训方法*
- [ ] **我讲你听**:讲解工作流程和标准
- [ ] **你讲我听**:验收学习成果
- [ ] **我做你看**:示范操作流程
- [ ] **你做我看**:检验实际操作

**2) 开市环境及安全检查**

*标准流程*
- [ ] 检查开市各板块的准备工作是否完成并达到标准
- [ ] 查看能源情况(电、气余额及余量)
- [ ] 检查电路、网络、电源插座是否正常无安全隐患,发现问题现场处理
- [ ] 检查设备是否正常运转(参照设备卡),异常设备立即维修或启用备用方案

*关键点*
- [ ] 要检查是否正常工作
- [ ] 检查有没有安全隐患
- [ ] 设备出现故障立即上报,及时解决;短时间内不能解决的,采用应急方案并向店总报备

**3) 产品货物情况检查**

*标准流程*
- [ ] 检查冰箱内食品是否生熟分离,温度是否正常
- [ ] 进行原料效期检查,是否变质/异味
- [ ] 检查原料/辅料是否充足

*关键点*
- [ ] 检查小料与吧台材料是否变质及异物
- [ ] 检查原材料保存是否完好，有无漏气、涨袋、变质
- [ ] 同时检查效期

**4) 收货验货**

*标准流程*
- [ ] 店长、厨师长或者在店管理双签
- [ ] 清点货物数量、质量、核对单据与价格
- [ ] 不符合标准则按退货流程执行并记录

#### 餐段流程

**1) 餐前准备**

*工作进度了解*
- [ ] 开市前对各岗位工作情况进行确认
- [ ] 协调各岗位的餐前准备工作

*工作验收及安排*
- [ ] 确认开餐准备情况
- [ ] 检查产品沽清情况,确保正常开市
- [ ] 如遇特殊情况则进行临时协调

**2) 餐中服务与管理**

*巡场管理*
- [ ] 巡场观察客人用餐情况并判断用餐满意度
- [ ] 进行桌访预防客诉问题发生
- [ ] 观察各岗位人员工作情况
- [ ] 防止窜岗、脱岗情况发生
- [ ] 确保工作有序进行
- [ ] 了解排队情况，根据排队情况调整门店翻台速度

*服务提醒*
- [ ] 提醒服务员巡台服务与数据工作
- [ ] 提醒后勤传菜人员划单、撤空盘
- [ ] 提醒进行重点产品与品牌文化介绍

*运营活动管理*
- [ ] 负责运营活动执行进度与成果
- [ ] 收藏打卡
- [ ] 评价引导
- [ ] 买单引导
- [ ] 产品推荐
- [ ] 会员活动

*突发事件处理*
- [ ] 客诉处理
- [ ] 能源中断
- [ ] 设备故障
- [ ] 安全问题

*关键点*
- [ ] **每日指标**:大众点评日新增好评数量不低于就餐桌数的15%，收藏打卡日新增数量不低于就餐人数的30%
- [ ] **平台警告期**:评价降低到日均就餐桌数5%，收藏打卡提升到不低于就餐人数的40%
- [ ] **警告期时长**: 通常为7-10天，期间需加强数据运营和服务质量管理
- [ ] **线上引导**:引导客人购买代金券及团购产品,增加线上交易热度、访问量提升榜单排名
- [ ] **私域转化**:提升私域用户量
- [ ] **客诉流程**:出现客诉情况,需根据客诉情况第一时间进行处理

**3) 餐后管理**

*标准流程*
- [ ] 安排进行餐后预收市或预打烊的工作
- [ ] 确保能源关闭,节能减耗
- [ ] 就餐、休息

*关键点*
- [ ] 用餐末期安排值班人员，防止人员松散影响门店形象与体验
- [ ] 预防工作拖延

#### 闭店流程

**1) 闭店检查**

*设备设施检查*
- [ ] 抽排检查
- [ ] 空调检查
- [ ] 灯光照明检查
- [ ] 冰柜检查
- [ ] 其他设施设备检查

*能源关闭*
- [ ] 水电关闭
- [ ] 燃气关闭

*区域环境卫生*
- [ ] 区域环境卫生(地面、墙面、操作台面)
- [ ] 设施设备卫生
- [ ] 日常物品卫生

*物资检查*
- [ ] 用具、清洁用具清洗/消毒,按4D标准摆放
- [ ] 日常物料储存收纳
- [ ] 冰箱内物品是否分类、分区、分层、分颜色摆放整齐
- [ ] 检查存放的原料有无变质现象

**2) 门店复盘**

*标准流程*
- [ ] 记录并总结今日门店经营与人员问题
- [ ] 自主进行记录或召开短会
- [ ] 将复盘内容上传至AI系统:
  - [ ] 登录SmartIce运营系统
  - [ ] 填写经营与复盘内容(营业情况、人员问题、经营问题、改进措施)
  - [ ] 上传相关资料(文字、图片或语音)
  - [ ] 提交复盘记录

*关键点*
- [ ] 每天关注人员训练和工作成果
- [ ] 及时发现问题并纠正引导
- [ ] 特别关注人员状态以及一些不规范的行为习惯
- [ ] 复盘记录必须在闭店后1小时内上传至AI系统
- [ ] 复盘内容要真实、具体、有针对性

**3) 闭店汇报**

*标准流程*
- [ ] 三关一闭检查(水关、电关、燃气关、门闭锁)
- [ ] 进行闭店情况及营业数据汇报
- [ ] 填制收银日报、优免明细表,发送汇总群
- [ ] 通过公司系统进行闭店汇报:
  - [ ] 登录SmartIce运营系统
  - [ ] 进入"闭店汇报"模块
  - [ ] 填写汇报内容:
    * 营业数据(营业额、就餐桌数、人数、客单价)
    * 经营指标(新增好评数、收藏打卡数、私域转化)
    * 人员情况(出勤人数、请假人数、培训情况)
    * 安全情况(安全检查结果、隐患排查、整改记录)
    * 客诉情况(客诉记录、处理结果、改进措施)
    * 其他事项(特殊情况说明、明日注意事项)
  - [ ] 上传相关图片(营业数据截图、问题现场照片等)
  - [ ] 提交汇报记录

*关键点*
- [ ] 店长确保闭店及卫生、安全工作正常
- [ ] 闭店汇报必须在闭店后1小时内完成
- [ ] 汇报内容要真实、完整、准确
- [ ] 重大问题需电话向上级汇报

### 2.2 前厅主管工作流程

#### 岗位职责

协助店长进行日常运营工作,包含工作分配与协调,人员培训与团队建设,对部分经营指标、顾客满意度以及团队满意度负责。

> **说明**: 前厅主管的工作流程与店长基本一致,主要区别在于权限和职责范围。具体流程参照店长工作流程执行,关键差异点如下:

#### 主要职责差异
- [ ] 协助店长进行门店餐前、餐中、餐后的事项安排与执行工作
- [ ] 开展班前会并统计到岗人数(店长组织复盘总结)
- [ ] 有计划地进行人员带训(店长负责整体培训规划)
- [ ] 店长不在时,代行店长职责,负责门店全面管理(具体权限范围见下文)
- [ ] 每日向店长汇报工作进展、问题及需要支持的事项

#### 工作流程要点
- [ ] **开店流程**: 参照店长开店流程执行,重点关注人员管理和带训工作
- [ ] **餐段流程**: 参照店长餐段流程执行,重点关注巡场管理和服务提醒
- [ ] **闭店流程**: 参照店长闭店流程执行,主管闭店时需完成以下工作:
  1. 完成闭店检查(参照店长闭店检查标准)
  2. 完成门店复盘(参照店长门店复盘标准)
  3. 通过公司系统进行闭店汇报:
     - 登录SmartIce运营系统
     - 进入"闭店汇报"模块
     - 填写汇报内容(营业数据、经营指标、人员情况、安全情况、客诉情况、其他事项)
     - 上传相关图片
     - 提交汇报记录
  4. 确保所有检查项完成
  5. 重大问题电话向店长汇报

#### 主管闭店特殊要求
- [ ] 主管闭店时,承担与店长同等的闭店责任
- [ ] 闭店汇报必须在闭店后1小时内完成
- [ ] 汇报内容要真实、完整、准确
- [ ] 次日需向店长当面汇报前一日闭店情况

#### 主管代行店长职责权限范围

**店长不在时,主管可行使以下权限**:
- [ ] 日常运营决策:人员调配、岗位安排、现场问题处理
- [ ] 客诉处理:500元以内的优免、赔偿决策(超过需报店长批准)
- [ ] 人员管理:请假审批(病假3天以内)、临时调班安排
- [ ] 物资采购:日常小额采购(单次500元以内)
- [ ] 安全处理:一般安全事件现场处理(重大事件立即报店长和店总)

**需报店长批准事项**:
- [ ] 超过权限范围的客诉处理
- [ ] 员工辞退、严重违纪处理
- [ ] 大额采购、维修、合同签订
- [ ] 营销活动变更、价格调整
- [ ] 重大安全事故

**主管日常工作汇报要求**:
- [ ] 每日闭店后通过系统提交工作汇报
- [ ] 次日上班时向店长当面汇报重要事项
- [ ] 遇重大问题立即电话向店长汇报
- [ ] 每周向店长提交一周工作总结

## 三、服务岗位

### 3.1 收银岗位

#### 岗位职责

负责客人下单、买单工作,并同时引导团购代金券及私域会员活动。

**详细职责**:
- [ ] 执行店长及前厅主管的工作安排
- [ ] 严格遵守财务管理制度,妥善保管现金、票据
- [ ] 熟练掌握吧台产品的名称、价格
- [ ] 根据顾客要求开发票
- [ ] 熟悉收款机操作技术和服务规范
- [ ] 熟悉线上线下优惠卡券使用规定
- [ ] 保持吧台整洁
- [ ] 低峰期协助前厅其他岗位

#### 餐段流程

**1) 餐前准备**

*自用物品检查*
- [ ] 通讯设备、工服、工具、收银设备检查
- [ ] 仪容仪表检查

*系统准备*
- [ ] 收银系统登录测试
- [ ] 打印机测试(打印纸充足)
- [ ] 验钞机测试、点菜机测试

*估清/限量预设*
- [ ] 根据厨房信息在系统中设置估清菜品
- [ ] 产品恢复后及时解除估清设置

*酒水与物资准备*
- [ ] 收银物品准备:
  - [ ] 检查并准备当日所需收银物品(热敏纸、发票、收款码等)
  - [ ] 前厅的客用茶水准备

**注**: 酒水清点工作已调整至水吧岗位负责,具体标准见水吧岗位流程。

*环境检查*
- [ ] 整理收银台桌面与卫生情况

**2) 餐中服务**

*预定接待*
- [ ] **电话预定标准流程**:
  1. 响铃3声内接听
  2. 自报门店:"您好,XX餐厅,很高兴为您服务,请问有什么可以帮到您?"
  3. 确认预定信息:询问称呼、用餐人数、用餐时间
  4. 安排座位并告知:"好的X先生/女士,已为您预留X人桌,X点到店即可"
  5. 确认特殊要求:"请问有什么特殊需求吗?(忌口、儿童座椅、包间等)"
  6. 核实信息:"再次确认一下,X先生/女士,X位,X点,对吗?"
  7. 记录手机号:"请留一下您的联系方式,方便我们为您保留座位"
  8. 礼貌致谢:"好的,期待您的光临,再见"
  9. 顾客先挂断电话
  10. 登记在预定登记本上,并同步给门店人员(通过系统或微信群通知)

- [ ] **微信预定标准流程**:
  1. 10分钟内回复
  2. 确认人数及到店时间
  3. 告知已预留座位
  4. 询问特殊需求
  5. 到店前1小时提醒确认
  6. 同步预定信息给门店人员

- [ ] **现场预定标准流程**:
  1. 询问预定信息(时间、人数)
  2. 可带领参观环境
  3. 登记预留桌号
  4. 留联系方式
  5. 同步预定信息给门店人员

- [ ] **折扣/免单预定**:
  1. 确认折扣/免单权限来源
  2. 店长签字确认
  3. 填写折扣登记表
  4. 预留座位并标注
  5. 同步预定信息给门店人员

*订单管理*

- [ ] 需要等位:"您好,现在客人比较多,需要稍等X分钟左右,我帮您取个号,您可以在等位区稍作休息"

  1. 叫号3次客人未到视为过号
  2. 过号后继续叫下一桌
  3. 过号客人回来后顺延3桌安排
  4. 过号客人如不接受顺延,需重新取号
  5. 向过号客人解释:"不好意思,您过号了,现在可以为您顺延3桌,大约X分钟,可以吗?"

  1. 问候:"您好,了解一下我们的餐厅吗?"

  - 准备菜单、活动宣传单页

  - 保持热情但不过分热情,避免让客人反感
  - 注意观察客人反应,适时调整话术

- [ ] **送客时机**:

  - 不催促客人离开

- [ ] 关注客人用餐情况

*餐中服务标准*
- [ ] **锅底熬开后流程**:打沫→分汤→介绍菜品→少量下菜→烫涮菜品→分菜
- [ ] **下煮菜**:涮烫菜品放桌上,其他菜品用相应餐具下锅
- [ ] **加汤**:锅内汤汁下降至LOGO线下及时添加至8分满
- [ ] **调火**:随时观察火力大小,涮烫菜品时调大,用餐缓慢时调小保持微开
- [ ] **搅锅**:长时间煮菜时及时搅动,画倒8,轻搅锅底
- [ ] **整理桌面**:下菜完毕后及时撤离空盘
- [ ] **添茶倒酒**:茶水量少时及时续水至7分满,主动询问是否需要倒酒
- [ ] **二次推销**:根据用餐情况向顾客推荐菜品
- [ ] **餐尾服务**:征询顾客同意后关火,送上牙签牙线等增值物品

*收台/复台标准*
- [ ] **收台时长**:小桌3分钟,大桌5分钟
- [ ] **收台顺序**:关火检查炉具→倒垃圾→玻璃器皿→茶杯→菜盘→筷子/汤漏勺→油碟碗→锅底
- [ ] **三轻**:轻拿、轻放、轻端
- [ ] **擦台标准**(5步骤):
  1. 用刮刀将桌面残渣刮入垃圾桶
  2. 用喷壶将清洁剂均匀喷洒在桌面(洗洁精:水=1:9或2:8)
  3. 用蓝色毛巾折叠成长条状擦拭(去除油污和污渍)

- [ ] **复台标准**:参照摆台标准,检查桌面物品是否齐全,卫生是否达标

- **冷饮**: 0-5℃
- **热饮**: 60-70℃
- **冰箱储存**: 冷藏0-5℃、冷冻-18℃以下
- **茶水准备**: 夏季提供冰水和常温水，冬季提供热水和温水

#### 容量标准快速查询
- **茶水**: 7分满
- **加汤**: 8分满
- **酒水斟倒**: 客人根据自己习惯倒酒即可

#### 比例标准快速查询
- **84消毒液**: 1:9(1毫升消毒液9毫升水)
- **清洁剂**: 洗洁精:水=1:9或2:8
- **备货量**: 1:4(每4张桌备1套)

#### 安全标准快速查询
- **指甲长度**: 不超过指尖2mm
- **迎宾送客**: 微笑并用挥手手势传递友善
- **物资离地**: 10cm以上
- **三关一闭**: 水关、电关、燃气关、门闭锁
