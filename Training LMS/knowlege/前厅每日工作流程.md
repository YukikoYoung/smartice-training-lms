# 前厅每日工作流程

> 基于时间主线的前厅整体运营流程
> 版本号: VN202501-WORKFLOW
> 制定日期: 2025-10-27

---

## 文档说明

本流程以时间为主线，整合前厅各岗位的每日工作安排，明确各时段的工作重点、岗位职责及协同要求，确保前厅运营高效有序。

**适用范围**: 餐厅前厅全岗位

---

### 营业时间说明

#### 工作日营业时间
- **上班时间**: 10:00
- **午市营业**: 11:30-14:00（2.5小时）
- **午休时间**: 14:00-17:00（3小时）
- **晚市营业**: 17:00-21:00（4小时）
- **停止接待**: 21:00
- **闭店时间**: 约21:30-22:00

#### 周末/节假日营业时间
- **上班时间**: 10:00
- **午市营业**: 11:30-14:00（2.5小时）
- **午休时间**: 14:00-16:30（2.5小时）
- **晚市营业**: 16:30-21:30（5小时，提前半小时开始，延长半小时结束）
- **停止接待**: 21:30
- **闭店时间**: 约22:00-22:30

> **说明**：周末和节假日晚市提前半小时开始接待（16:30），延长半小时结束（21:30），以应对更大的客流量。

---

**使用建议**:
- 各岗位人员应熟记本岗位在各时段的工作职责
- 管理人员应掌握全流程，做好统筹协调
- 周末/节假日注意时间调整，提前做好准备
- 遇特殊情况灵活调整，但基本流程不变

---

## 岗位总览

### 管理岗位
- **店长**: 全面管理门店运营，统筹协调各岗位工作
- **前厅主管**: 协助店长管理前厅，重点负责人员带训和现场服务管理

### 服务岗位
- **收银员**: 预定管理、订单处理、买单服务、寄存服务
- **迎宾员**: 迎宾送客、排队叫号、等位区服务、店推
- **服务员**: 带客入座、餐中服务、巡台加汤、复台摆台
- **水吧员**: 饮品制作、酒水管理、原料管理
- **传菜员**: 菜品传递、小料准备、空盘收取
- **保洁员**: 快速收台、洗手间清洁、公共区域卫生

## 每日工作时间线

### 阶段一：早晨准备（10:00-11:30）

> **时间说明**：工作日和周末/节假日的准备时间相同，均为10:00-11:30，确保11:30准时开始午市营业。

#### 10:00-10:15 到岗准备

**所有岗位 - 个人准备**
- [ ] 准时到岗，更换工服
- [ ] 仪容仪表自检（参照仪容仪表标准）
- [ ] 工服整洁、工牌佩戴规范
- [ ] 检查个人工具是否齐全

**管理岗位 - 早到检查**
- **店长/主管**:
  - [ ] 提前20-30分钟到店（即9:30-9:40到店），确保在员工10:00到岗前完成基础检查
  - [ ] 检查收市卫生是否达标
  - [ ] 检查能源情况（电费、燃气余额）
  - [ ] 检查设备运转是否正常
  - [ ] 检查冰箱温度及食材状况
  - [ ] 发现问题立即处理或安排应急方案

> **协同要点**: 店长/主管需在员工到岗前完成基础检查，发现问题及时处理，确保开市顺利。

---

#### 10:15-10:45 岗位准备

> **时间调整说明**：本时段为各岗位准备核心时段，从原15分钟延长至30分钟，确保各岗位能够从容完成所有准备工作，特别是新员工和设备较多的岗位。

**收银岗位**
- [ ] 收银系统登录测试
- [ ] 打印机、验钞机、点菜机测试
- [ ] 检查打印纸、发票是否充足
- [ ] 根据厨房信息设置估清菜品
- [ ] 准备收银物品（热敏纸、收款码等）
- [ ] 整理收银台桌面与卫生

**迎宾岗位**
- [ ] 对讲机测试（频道16，已于前一日充电）
- [ ] 取号机测试出票
- [ ] 音响连接测试（叫号语音）
- [ ] 电视开启并切换至排号界面
- [ ] 等位区桌椅摆放整齐
- [ ] 摆放广告物料（菜单、活动海报、立牌）
- [ ] 准备等位区小吃（3-5种，周末数量增加到1.5-2倍，即准备到平时的150%-200%）
- [ ] 准备等位区茶水（夏季冰水+常温水，冬季热水+温水）
- [ ] 准备日常物料（预定本、笔、纸巾、薄荷糖、牙签、雨衣、橡皮筋）
- [ ] 准备增值物品（除味喷雾、充电线、WIFI密码卡）
- [ ] 检查门口玻璃门窗，有污渍立即擦拭
- [ ] 检查门口水牌、展架摆放

**服务岗位**
- [ ] 对讲机测试
- [ ] 检查电磁炉（每台逐一测试加热功能）
- [ ] SmartIce助手系统检查（登录测试、网络测试）
- [ ] 备餐柜物资准备（餐具按1:2比例备货，即1张台备2套餐具；高峰时段或周末按1:3比例，即1张台备3套餐具，确保快速翻台时有充足餐具）
- [ ] 准备日常物料（小票夹、发圈、眼镜布、手机架）
- [ ] 准备客用物料（餐具、纸巾、茶壶、茶杯、筷筒、围裙）
- [ ] 准备服务用茶水（按预估客流准备充足茶水）
- [ ] 按标准摆台（餐具五件套、筷筒、汤勺漏勺、纸巾、围裙）
- [ ] 水牌、宣传物料摆放
- [ ] 提前预估所需汤底量，各区域备好加汤汤底
- [ ] 检查桌面、椅面、地面卫生（无杂质、油污、水渍）

**水吧岗位**
- [ ] 仪容仪表检查（佩戴口罩、发帽）
- [ ] 领货验收，核对数量检查质量
- [ ] 检查原材料，先备紧要主料，再备辅料
- [ ] 检查设备（制冰机、开水机、冰箱、酒水柜）
- [ ] 冰箱温度检查（冷藏0-5℃，冷冻-18℃）
- [ ] 原料效期检查，临期优先使用，过期报损
- [ ] 准备主料（各种基底、茶汤）
- [ ] 准备辅料（水果、装饰、吸管、杯子）

**传菜岗位**
- [ ] 对讲机测试，仪容仪表检查（手套、口罩、发帽）
- [ ] 档口清洁
- [ ] 小料准备（按备货量表）
- [ ] 餐框、餐车、托盘准备
- [ ] 出品台台面整洁，地面无水渍油渍
- [ ] 收台清洁工具准备

**保洁岗位**
- [ ] 对讲机测试，仪容仪表检查
- [ ] 餐框、餐车、收餐盒清洁，按4D标准摆放
- [ ] 洗手间物品补充（卷纸、垃圾袋）
- [ ] 梳妆台增值物品补充
- [ ] 收餐车/餐框按区域摆放
- [ ] 垃圾桶清洁，更换垃圾袋（无过夜垃圾）
- [ ] 洗手间及公共区域清洁达标

> **协同要点**:
> - 各岗位同步进行准备工作，不窜岗、不拖延
> - 迎宾、服务员需互相协助完成厅面摆台和环境检查
> - 水吧、传菜员需提前沟通当日重点菜品的备料需求
> - 保洁员完成本职工作后，协助其他岗位完成准备

---

#### 10:45-11:05 管理检查与协调

> **时间调整说明**：配合岗位准备时间延长，管理检查时段相应后移，但检查内容和标准保持不变。

**店长/主管 - 全面巡检**
- [ ] 检查各岗位准备工作进度和质量
- [ ] 核实人员到岗情况，了解请假/缺勤
- [ ] 检查收市卫生是否达标，不合格立即整改
- [ ] 检查电路、网络、电源插座安全
- [ ] 检查设备运转情况（参照设备卡）
- [ ] 检查冰箱内食品生熟分离、温度正常
- [ ] 检查前一天剩余食材是否变质
- [ ] 检查原料/辅料是否充足
- [ ] 所有问题必须在开市前完成整改
- [ ] 无法整改的问题向上级报备应急方案

**收货验货**（如有到货）
- [ ] 店长/厨师长双签
- [ ] 清点货物数量、质量、核对单据与价格
- [ ] 不符合标准按退货流程执行并记录

> **协同要点**:
> - 店长/主管需系统性巡检，不遗漏任何环节
> - 发现问题当场指导整改，给出具体解决办法
> - 协调各岗位之间的准备进度，确保同步完成

---

#### 11:05-11:30 开市准备与最后确认

> **时段定位**：11:05-11:20为开市最后准备和调整时间，11:20-11:30为最后确认时段。

❗**关键控制点**：本时段是开市前最后检查，必须确保所有准备工作100%完成，任何遗漏都会影响营业。午市11:30准时开始接待客人。

**11:05-11:20 最后准备**（如岗位准备已完成可提前进入待命状态）
- [ ] 各岗位完成收尾工作
- [ ] 调整站位，进入待命状态

**11:20-11:30 最后确认**

**所有岗位 - 自检确认**
- [ ] 各岗位自检准备工作完成情况
- [ ] 确认物资、设备、卫生全部达标
- [ ] 待命准备迎接第一批客人

⚠️ **易错提醒**：
- 对讲机/取号机是否充满电（前一日应充电）
- 估清菜品是否已在系统中设置
- 等位区小吃、茶水是否准备充足
- 服务用茶水是否备足

**店长/主管 - 开市确认**
- [ ] 最后一轮巡检，确认开餐准备完成
- [ ] 确认产品沽清情况
- [ ] 对讲机通知："各岗位准备完毕，准备开市"
- [ ] 如遇特殊情况进行临时协调

**收银岗位**
- [ ] 系统再次确认，估清菜品核对
- [ ] 待命接听预定电话

**迎宾岗位**
- [ ] 站位准备（距门口2-3米处）
- [ ] 保持良好站姿和微笑
- [ ] 准备迎接第一批客人

> **协同要点**:
> - 全员进入待命状态，做好迎客准备
> - 各岗位通过对讲机保持沟通
> - 店长/主管最后确认，给出开市指令

**✓ 本阶段完成确认**：
- ✓ 所有岗位准备工作100%完成
- ✓ 店长/主管巡检通过，无遗漏问题
- ✓ 对讲机频道16通讯正常，全员待命
- ✓ 开市指令已发出，准备迎接第一批客人

### 阶段二：午市营业（11:30-14:00）

> **阶段转换说明**：从准备阶段到营业阶段，是从"幕后"到"台前"的关键转换。此时所有准备工作必须已100%完成，全员从准备状态切换到服务状态，以最佳状态迎接第一批客人。

> **时间说明**：工作日和周末/节假日的午市营业时间相同，均为11:30-14:00（2.5小时）。

#### 11:30-12:30 午市开始

> **时段定位**：午市预热期，客流从11:30后逐步增加。此时段要在保证服务质量的前提下，为即将到来的高峰期做好节奏调整。

**迎宾岗位 - 迎客与引导**
- [ ] 站位迎客（距门口2-3米，保持微笑）
- [ ] 看到客人主动迎接："您好，欢迎光临XX餐厅"
- [ ] 询问人数："请问几位用餐呢？"
- [ ] 判断安排：
  - 有位：直接带位入座
  - 无位：安排取号等位
- [ ] 预定客人：确认姓氏、人数、时间后带位
- [ ] 取号叫号（客流高峰时）：
  - 根据人数取号
  - 告知等位时间
  - 引导至等位区
  - 与服务组确认空桌后按序叫号
  - 过号处理：叫3次未到视为过号，回来后顺延3桌
- [ ] 带客入座标准：
  - 根据人数安排合适桌位
  - 优先安排靠窗、视野好的位置
  - 有老人/小孩优先安排方便进出的位置
  - 手势引导："这边请"
  - 行走速度与客人同步
  - 走在客人前方1-2步
  - 遇台阶提醒"小心台阶"
  - 协助老人/小孩/女性入座
- [ ] 向服务员交接桌号和人数，确认接手后离开

**等位区服务**（客流高峰时）
- [ ] 引导等位客人："等位区在这边，有免费小吃和茶水"
- [ ] 介绍便利设施："WIFI密码在桌上，这边有充电口"
- [ ] 提前2-3桌提醒客人准备
- [ ] 保证小吃、茶水充足，定期巡视补充
- [ ] 保持等位区整洁，及时清理桌面杂物
- [ ] 顾客解答：取号规则、活动介绍、菜品推荐
- [ ] 等位时间长时主动告知进度，安抚客人

**店推服务**（客流少时）
- [ ] 主动对路过客人店推
- [ ] 门口张望客人主动介绍
- [ ] 店推话术：
  1. "您好，了解一下我们的餐厅吗？"
  2. "我们是XX餐厅，主打非遗酸汤锅底"
  3. "现在有优惠活动，XX折扣/赠品"
  4. "要不要进来看看菜单？"
  5. 客人拒绝时礼貌告别："好的，有需要随时欢迎您"

**送客服务**（贯穿全天）
- [ ] 观察客人买单后准备离开，主动上前送客
- [ ] 主动询问："您好，用餐满意吗？"
- [ ] 引导至门口："您好，这边请，我送您出去"
- [ ] 温馨提示："小心台阶"（如有台阶）
- [ ] 天气提醒：
  - 下雨：提供雨衣，"外面在下雨，这是雨衣，雨天地滑请注意安全"
  - 炎热：提醒"外面比较热，注意防暑"
- [ ] 告别：微笑挥手"慢走，欢迎下次光临"
- [ ] 送客增值服务：
  - 主动询问是否需要打包（与服务员配合）
  - 提供除味喷雾
  - 引导线上好评（体验满意的客人）
- [ ] 送客至门口，目送客人离开

**收银岗位 - 预定与订单**
- [ ] 接听预定电话（3声内接听）：
  1. "您好，XX餐厅，很高兴为您服务"
  2. 确认预定信息（称呼、人数、时间）
  3. 安排座位并告知
  4. 询问特殊需求
  5. 核实信息
  6. 记录手机号
  7. 礼貌致谢
  8. 等客人先挂断
  9. 登记预定本，同步门店人员
- [ ] 微信预定（10分钟内回复）
- [ ] 现场预定登记
- [ ] 折扣/免单预定需店长签字
- [ ] 负责客人加单、退单、活动赠送
- [ ] 观察点菜情况，发现异常及时沟通
- [ ] 浏览优惠券核销，避免差错
- [ ] 协助现场顾客咨询

**服务岗位 - 餐中服务**
- [ ] 带客入座后服务流程：
  1. 斟倒茶水（7分满）
  2. 介绍点餐方式（扫码/服务员点餐）和团购使用
  3. 介绍特色产品（锅底、牛肉、云贵精选菜、手作甜品）
  4. 介绍小料台位置及特色蘸料
  5. 询问忌口，推荐畅销品和促销品
  6. 提供增值物品（围裙、眼镜布、头绳、婴儿椅）
- [ ] 活动推荐：第一时间推荐门店活动
- [ ] 增减餐位：
  - 减位顺序：座椅→茶杯→餐具→调整间距
  - 加位顺序：调整间距→餐具/茶杯→座椅
- [ ] 上菜服务：
  - 上菜前挪位、撤空盘
  - 建议顺序：锅底→荤菜→素菜
  - 介绍特色菜品、烫涮时间
  - 手指不触碰菜品
- [ ] 餐中服务：
  - 锅底熬开：打沫→分汤→介绍→少量下菜→烫涮→分菜
  - 加汤：汤汁降至LOGO线下及时添加至8分满，"您好，帮您加一下汤，小心烫"
  - 调火：涮烫时调大，用餐缓慢时调小保持微开
  - 搅锅：长时间煮菜时及时搅动，画倒8
  - 整理桌面：及时撤离空盘
  - 添茶倒酒：茶水少时续至7分满
  - 二次推销：根据用餐情况推荐菜品
- [ ] 巡台服务（定期）：
  - 加汤、加茶水
  - 调整电磁炉档位
  - 关注客人用餐情况
  - 及时撤走空盘
- [ ] 活动推广：
  - 主动推荐店内活动
  - 餐中根据桌面广告物料推荐
  - 收藏打卡及评价引导
  - 团购活动推荐

**传菜岗位 - 出餐服务**
- [ ] 核对出餐小票，检查菜品符合十不出品标准：
  1. 餐具破损不传
  2. 盘饰不符不传
  3. 菜品异味不传
  4. 菜品异物不传

> **阶段转换说明**：从营业状态转入收市状态，标志着一天营业的结束。此阶段虽然劳累但至关重要，收市工作的质量直接影响第二天的开市准备。要克服疲惫，按标准完成所有收市工作，确保门店安全。

## 异常情况处理指引

### 1. 突然停电

**立即行动**：
- [ ] 管理岗位立即启动应急照明（手电筒、应急灯）
- [ ] 收银员切换至备用收银方式（手机收款码）
- [ ] 服务员安抚客人："很抱歉，暂时停电，我们正在处理"
- [ ] 水吧员停止制作，保护设备
- [ ] 管理岗位联系物业/电力公司确认恢复时间

**持续时间判断**：
- **15分钟内恢复**：维持营业，手工记账，电力恢复后补录系统
- **15-30分钟**：暂停新客入座，完成在店客人服务
- **30分钟以上**：建议暂停营业，妥善处理在店客人（菜品打折/免单）

**恢复后检查**：
- [ ] 检查所有设备是否正常（特别是冰箱温度）
- [ ] 检查食材是否受影响
- [ ] 收银系统数据补录

---

### 2. 网络/系统故障

**立即行动**：
- [ ] 收银员切换至备用收银方式（扫码枪、手机）
- [ ] 手工记录订单和金额
- [ ] 服务员告知客人："系统维护中，仍可正常点餐买单"
- [ ] 管理岗位联系技术支持

**应对措施**：
- [ ] 启用纸质菜单点餐
- [ ] 手工记录订单，厨房凭手写单出餐
- [ ] 买单时手工计算，客人扫码直接支付
- [ ] 系统恢复后补录所有订单数据

---

### 3. 关键设备故障

**电磁炉故障**：
- [ ] 立即停用故障电磁炉，换至备用桌位
- [ ] 服务员提前告知客人并致歉
- [ ] 联系维修，预估修复时间
- [ ] 如多台故障，考虑限制接待（只开正常桌位）

**收银系统故障**：
- [ ] 切换至备用收银设备（备用电脑/平板）
- [ ] 使用手机收款码应急
- [ ] 手工记账，系统恢复后补录

**空调故障**：
- [ ] 夏季：启用备用风扇，向客人提供冰水、冰巾
- [ ] 冬季：启用备用取暖设备
- [ ] 联系维修，评估是否影响营业

**营业中缺岗**：
- [ ] 立即评估影响，及时调配人员补位
- [ ] 管理岗位亲自上场，协助薄弱环节
- [ ] 必要时向客人说明情况，争取理解

**关键岗位应急预案**：
- 店长/主管缺岗：另一方代行职责，重大事项电话请示
- 收银员缺岗：主管兼任，或培训其他员工应急顶岗
- 厨师长缺岗：限制菜单，只做简单菜品

---

### 5. 客流超预期爆满

**判断标准**：
- 等位超过30组
- 预估等位时间超过1小时
- 厨房出餐速度明显跟不上

**应对措施**：
- [ ] 管理岗位评估承载能力，决定是否暂停叫号
- [ ] 迎宾员如实告知客人等位时间，让客人自主选择
- [ ] 加快翻台：服务员、保洁员提速，缩短收台时间
- [ ] 简化服务流程：保证核心服务，适当简化非必要环节
- [ ] 厨房提速：优先出畅销品，暂停复杂菜品
- [ ] 管理岗位现场指挥调度，全员补位

**底线原则**：
- 宁可暂停接待，不可降低服务质量导致客诉
- 对等位客人诚实告知，避免过度承诺

---

### 6. 大量客诉集中爆发

**判断标准**：
- 1小时内出现3次以上客诉
- 出现严重客诉（食品安全、服务态度恶劣）

**立即行动**：
- [ ] 管理岗位亲自处理所有客诉
- [ ] 分析客诉原因：
  - 菜品问题→立即通知厨房停止相关菜品
  - 服务问题→立即纠正，加强现场管理
  - 环境问题→立即整改
- [ ] 召开紧急短会，全员提高警惕
- [ ] 加强巡台，主动发现问题，提前处理

**预防措施**：
- [ ] 管理岗位加强巡场，及时发现服务短板
- [ ] 加强与客人沟通，主动询问满意度
- [ ] 服务员提高服务标准，避免因忙乱降低质量

---

### 7. 食品安全紧急情况

**客人反映食材有异物/变质**：
- [ ] 立即停止该菜品供应
- [ ] 封存问题菜品和同批次原料
- [ ] 向客人诚恳道歉，免单处理
- [ ] 立即检查厨房，查找原因
- [ ] 拍照记录，上报店长和公司
- [ ] 做好客人安抚，避免扩大影响

**客人出现疑似食物中毒**：
- [ ] 立即停止营业，封存所有食材
- [ ] 拨打120，协助客人就医
- [ ] 立即上报店长、区域经理、总公司
- [ ] 保留现场证据，配合调查
- [ ] 绝不隐瞒，积极配合处理

---

### 8. 火灾/安全事故

**发现火情**：
- [ ] 立即按下火警按钮
- [ ] 使用灭火器灭火（小火）
- [ ] 大火立即疏散客人，拨打119
- [ ] 切断电源、关闭燃气
- [ ] 管理岗位组织疏散，确保人员安全
- [ ] 人员安全是第一位，财物第二位

**客人受伤**：
- [ ] 立即停止营业，救助伤者
- [ ] 严重伤情拨打120
- [ ] 保护现场，等待处理
- [ ] 立即上报公司，等待指示
- [ ] 做好记录，配合调查
