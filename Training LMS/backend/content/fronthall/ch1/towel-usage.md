# 毛巾使用标准

## 课程概述

本课程详细介绍餐厅前厅毛巾的分类使用、清洗消毒和管理标准。正确使用毛巾是保证卫生安全、避免交叉污染的重要环节,是所有前厅员工必须掌握的基本技能。

---

## 一、毛巾分类使用

### 1.1 为什么要分色使用?

毛巾按照使用场景进行颜色区分,主要目的是:
- **避免交叉污染**:不同清洁区域的污染程度不同,分色使用能防止高污染区的细菌传播到低污染区
- **提高卫生标准**:确保餐具、桌面等与客人直接接触的区域保持高清洁度
- **便于管理**:颜色标识清晰,员工不易混淆,管理者易于检查

### 1.2 前厅毛巾分类标准

| 颜色 | 使用区域 | 说明 |
|------|----------|------|
| **蓝色毛巾** | 桌面清洁(一次擦拭) | 用于清除桌面食物残渣、污渍 |
| **紫色毛巾** | 桌面抛光(二次擦拭) | 在蓝色毛巾清洁后使用,让桌面更光亮 |
| **白色毛巾** | 餐具擦拭 | 专门用于擦拭餐具,要求最高清洁度 |
| **棕色毛巾** | 洗手间清理 | 用于清洁洗手间台面、镜面等 |
| **绿色毛巾** | 椅凳清理 | 用于清洁椅子、凳子 |

### 1.3 使用原则

**严格分色使用**:
- 每种颜色的毛巾只能用于指定区域
- **严禁交叉使用**:例如,不能用蓝色毛巾擦餐具,不能用棕色毛巾擦桌面
- 定位摆放,拿取时注意颜色标识

**使用顺序**:
- 桌面清洁:先用蓝色毛巾擦拭一次,再用紫色毛巾抛光
- 餐具擦拭:只使用白色毛巾,且必须是清洗消毒后的干净毛巾

## 二、清洗及消毒标准

### 2.1 清洗流程

**第一步:初步清洗**
- 用洗洁精加热水清洗
- 充分搓洗,去除油污和污渍
- 用清水冲洗干净,确保无洗洁精残留

**第二步:消毒浸泡**
- 使用84消毒液进行消毒
- 消毒液与水的比例:**1:9**(1毫升消毒液 + 9毫升水)
- 浸泡时间:**30分钟**

**配制示例**:
- 10ml消毒液 + 90ml水 = 100ml消毒液
- 或:2瓶盖(约10ml)消毒液 + 90ml水

**第三步:冲洗晾干**
- 浸泡30分钟后,用清水充分冲洗
- 拧干水分,挂在指定位置晾晒
- 确保毛巾完全干燥后再使用

### 2.2 消毒注意事项

⚠️ **安全警示**:
- 84消毒液与洗洁精**不可混用**(会产生有毒氯气)
- 必须先用洗洁精清洗,冲洗干净后,再用84消毒液浸泡
- 消毒后必须用清水冲洗干净,避免消毒液残留刺激皮肤
- 操作时建议佩戴手套,避免皮肤直接接触消毒液

## 三、使用要求与时间节点

### 3.1 更换时间

**每日两次更换**:
- **第一次**:14:00~14:30(午市结束后)
- **第二次**:22:00~22:30(晚市结束后)

**目的**:保证毛巾在一天营业中保持卫生标准,避免长时间使用导致细菌滋生。

### 3.2 管理要求

**定位摆放**:
- 前厅毛巾统一存放在指定区域
- 分色分类管理,不同颜色毛巾分开放置
- 便于员工快速取用正确颜色的毛巾

**禁止裁剪**:
- 毛巾不得裁剪使用
- 保持毛巾完整性,便于清洗和消毒

**定期更换**:
- 每20天检查毛巾状态
- 对不符合要求的毛巾淘汰更换(如:破损、严重污渍、变硬等)

**晚市消毒**:
- 每日晚市结束后,将所有使用过的毛巾消毒浸泡30分钟
- 清洗后统一晾晒在指定位置
- 确保第二天开市时有足够干净毛巾使用

## 四、常见错误与纠正

### 4.1 常见错误

❌ **错误1**:用同一条毛巾擦桌子又擦餐具
- **危害**:桌面细菌污染餐具,影响食品安全
- **纠正**:严格分色使用,桌面用蓝色/紫色,餐具用白色

❌ **错误2**:毛巾长时间不更换
- **危害**:毛巾积累细菌,越擦越脏
- **纠正**:每日两次更换,发现毛巾脏污立即更换

❌ **错误3**:消毒液浓度不够或浸泡时间不足
- **危害**:消毒不彻底,细菌未被杀灭
- **纠正**:严格按1:9比例配制,浸泡满30分钟

❌ **错误4**:消毒后不冲洗就使用
- **危害**:消毒液残留刺激皮肤,污染食品
- **纠正**:消毒后必须用清水充分冲洗

### 4.2 自查清单

使用毛巾前,请自问:
- [ ] 我拿的毛巾颜色对吗?
- [ ] 这条毛巾是干净的吗?
- [ ] 毛巾有没有异味或明显污渍?
- [ ] 上次更换毛巾的时间是什么时候?

## 五、实际操作示例

### 场景1:翻台时清洁桌面

**步骤**:
1. 先用蓝色毛巾擦拭桌面,去除食物残渣和污渍
2. 将食物残渣丢入垃圾桶
3. 再用紫色毛巾进行二次擦拭,让桌面更加光亮
4. 检查桌面是否干净,无水渍、无污渍

### 场景2:擦拭餐具

**步骤**:
1. 确认餐具已经过清洗消毒
2. 取干净的白色毛巾
3. 擦拭餐具表面,确保无水渍、无污渍
4. 将擦拭好的餐具整齐摆放
5. 使用后的白色毛巾放入待洗区

### 场景3:清洁洗手间

**步骤**:
1. 使用棕色毛巾清洁洗手间台面、镜面
2. 清洁后的棕色毛巾**单独存放**,不与其他颜色毛巾混放
3. 及时清洗消毒,避免细菌传播

**课程总结**:

毛巾的正确使用看似简单,却关系到餐厅的卫生安全和客人健康。通过严格的分色使用、定期消毒、及时更换,我们能够有效避免交叉污染,确保餐厅环境和餐具的清洁度,为客人提供安全放心的用餐环境。

**记住**:蓝紫桌面、白色餐具、棕色洗手间、绿色椅凳,分色使用不混淆!
