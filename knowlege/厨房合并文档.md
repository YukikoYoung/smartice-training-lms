# 第一部分:通用管理标准

## 1.1 仪容仪表标准

#### 男士标准
- [ ] 发型:头发前不过眉、侧不过耳、后不过领,后厨要求短发
- [ ] 面容:干净整洁,不蓬头垢面,不留胡须,佩戴口罩
- [ ] 饰品:严禁佩戴夸张饰品
- [ ] 衣服/围裙:标准工服穿戴齐备、干净整洁、无异味,无皱褶,无油污
- [ ] 手部:指甲不得超过指尖,保持与指尖齐平或略短,干净整洁无污渍;指甲内无污垢;工作期间按要求佩戴一次性手套
- [ ] 裤子:统一按照工装要求,干净无皱褶,无异味,无油污
- [ ] 鞋袜:禁止拖鞋/凉鞋;袜子黑色或深色

#### 女士标准
- [ ] 发型:梳理整洁,先梳成马尾、再用发卷捆住或盘起;如有刘海不能遮住眉毛、不能有耳发;不允许披头散发
- [ ] 面容:干净整洁,可化淡妆、不允许浓妆、佩戴口罩
- [ ] 饰品:严禁佩戴夸张饰品
- [ ] 衣服/围裙:标准工服穿戴齐备、干净整洁、无异味,无皱褶
- [ ] 手部:指甲不得超过指尖,保持与指尖齐平或略短,干净整洁无污渍;不能做美甲/涂指甲,指甲内无污垢;工作期间佩戴一次性手套
- [ ] 裤子:统一按照工装要求,干净无皱褶,无异味,无油污
- [ ] 鞋袜:黑色鞋,禁止拖鞋/凉鞋;袜子黑色或深色

#### 七步洗手法
1. [ ] 掌心相对,手指并拢相互揉搓
2. [ ] 手心对手背沿指缝相互搓擦,交换进行
3. [ ] 掌心相对,沿指缝相互搓擦
4. [ ] 手指内收握拳,交换进行
5. [ ] 一手握另一手大拇指旋转搓擦,交换进行
6. [ ] 将五个手指甲并拢在另一手掌心旋转搓擦,交换进行
7. [ ] 螺旋式擦洗手腕,交换进行

### 1.2 毛巾使用标准

#### 分色使用规范
- [ ] **蓝色毛巾**:中高污染区域 - 烧菜台面清洁、小吃台面清洁、打锅间、员工餐台面
- [ ] **绿色毛巾**:中低污染区域 - 水吧操作台面、各岗位器具、设备清洁
- [ ] **白色毛巾**:低污染区域 - 餐具擦拭、包子操作台
- [ ] **棕色毛巾**:高污染区域 - 厨房地面

#### 清洗及消毒

**84消毒液稀释标准**
> 注: 市售84消毒液有效氯含量通常为5%,以下配比按5%计算

- [ ] **毛巾消毒配比**:
  - 比例: 1:100 (即1份消毒液+99份水)
  - 具体操作: 10ml消毒液 + 990ml水 = 1000ml溶液
  - 或简便方法: 2瓶盖(约10ml)消毒液 + 1瓶(1000ml)矿泉水瓶的水,再倒掉10ml
  - 有效氯浓度: 约500mg/L
  - 浸泡时间: 30分钟

**清洗流程**
- [ ] 洗洁精加热水清洗
- [ ] 84消毒液消毒:按上述配比浸泡30分钟后清水冲洗

**注意事项**
- [ ] 84消毒液与洗洁精不可混用(会产生有毒氯气)
- [ ] 消毒后必须用清水冲洗干净,避免残留
- [ ] 配制好的消毒液当天使用,不过夜(有效氯会挥发)
- [ ] 操作时必须佩戴手套,避免皮肤直接接触
- [ ] 保持通风,避免吸入刺激性气体

#### 使用要求
- [ ] 每日使用的毛巾需在午市结束后(约13:30-14:00)和晚市结束后(约21:30-22:00)各更换或清洁一次
- [ ] 定位摆放,不可交叉颜色使用
- [ ] 毛巾禁止裁剪,每20天对不符合要求的毛巾淘汰更换
- [ ] 非棕色毛巾不可用于地面或特殊区域
- [ ] 前厅后厨毛巾不混放,四色毛巾储存严禁混放(棕色毛巾单独存放)
- [ ] 每日晚市将所有毛巾消毒浸泡30分钟,清洗后统一晾晒在指定位置

### 1.3 4D现场管理标准

> **4D管理定义**:
> - **整理到位**: 区分要与不要,清除不要的物品
> - **责任到位**: 划分责任区,明确责任人
> - **培训到位**: 培训员工执行标准
> - **执行到位**: 检查考核,持续改进

#### 基本原则
- [ ] 区域内保持整洁卫生、工器具、原料等定位摆放,整齐有序
- [ ] 预制的半成品,应避免污染,与原料分开分类存放
- [ ] 食品按类别、品种分架,离地25cm以上、离墙5cm以上整齐摆放
- [ ] 散装食品及原料储存容器加盖密封,同时经常检查,防止霉变
- [ ] 需冷藏的食材,需加膜加盖放置于指定冰箱
- [ ] 加工工器具使用后及时清洗消毒,做到无霉、无锈、无渍、无残渣、定位摆放
- [ ] 操作时注意保持卫生,操作后及时清扫,保证地面、工作台面无异味、无残渣、无卫生死角
- [ ] 操作中产生的废弃物应边产生边处理,及时清理到有盖的容器内,做到一餐一清
- [ ] 完善防虫、防蝇、防鼠、防蟑螂、防尘设施,做到无虫、无蝇、无鼠、无蟑螂

### 1.4 食品安全标准

#### 健康检查规范
- [ ] 从业人员必须持《健康证》上岗
- [ ] 凡患有痢疾、伤寒等消化道传染病以及其他有碍食品安全疾病的,治愈前不得参加接触直接入口食品的生产经营

#### 个人卫生
- [ ] 仪容仪表符合规范
- [ ] 操作间不得有吸烟、饮食、随地吐痰等可能污染食物的行为
- [ ] 制作食品前洗手消毒,不得用手直接拿放熟食
- [ ] 严禁将与工作无关的任何物品带入操作间
- [ ] 员工进入操作间严格执行洗手消毒程序

#### 食品卫生

**食材新鲜度检查标准**:

**通用检查方法(适用所有食材)**:
- [ ] **看**:色泽是否正常(肉类鲜红、蔬菜翠绿、海鲜透明;发黑、发黄、有斑点为不新鲜)
- [ ] **闻**:是否有异味(酸味、臭味、氨水味为变质;正常食材应无异味或有食材本身清香)
- [ ] **摸**:弹性是否正常(肉类按压后能快速回弹、蔬菜脆挺不发软、海鲜肉质紧实)
- [ ] **查**:检查标签日期,是否在保质期内;检查包装是否完整无破损
- [ ] **处理**:发现不新鲜食材立即隔离,拍照记录,报告厨师长,不得使用

**分类新鲜度判断细则**:

**1. 肉类(牛肉、猪肉、羊肉等)**:
- [ ] **新鲜标准**:
  - 色泽:鲜红色或淡红色,有光泽,脂肪部分洁白或淡黄色
  - 气味:无异味,或有肉类特有的腥味(非臭味)
  - 弹性:用手指按压,肉质紧实,按压后立即回弹,不留凹痕
  - 表面:干爽,不发黏,无黏液
  - 纹理:肌肉纹理清晰,切面有湿润感但不出水
- [ ] **不新鲜特征**:
  - 色泽:发暗、发紫、发灰、有绿色斑点
  - 气味:有酸味、臭味、氨水味或其他异味
  - 弹性:按压后凹陷不回弹,或肉质松软无弹性
  - 表面:发黏、有黏液、表面湿漉漉
  - 出现以上任一特征,立即判定为不新鲜,不得使用

**2. 海鲜类(鱼、虾、蟹、贝类等)**:
- [ ] **鱼类新鲜标准**:
  - 眼睛:眼球饱满凸起,角膜透明清亮,眼珠黑亮
  - 鳃部:鳃片鲜红或粉红,黏液透明,无异味
  - 鱼鳞:紧贴鱼身,有光泽,不易脱落
  - 鱼肉:肉质紧实有弹性,按压后立即回弹
  - 气味:有海洋特有的鲜腥味,无腐臭味
- [ ] **鱼类不新鲜特征**:眼球凹陷、角膜浑浊、鳃片发黑或发褐、鳞片松脱、肉质松软、有臭味
- [ ] **虾类新鲜标准**:
  - 外壳:光亮透明,有弹性,颜色鲜艳(青虾青绿色、白虾透明白色)
  - 头部:头部与身体连接紧密,不脱落,虾头无发黑
  - 肉质:虾肉紧实,弯曲有弹性,尾部自然卷曲
  - 气味:有海鲜特有鲜味,无氨水味或腐臭味
- [ ] **虾类不新鲜特征**:外壳变红或发黑、头部松脱或发黑、虾肉松软、有氨水味或臭味
- [ ] **贝类新鲜标准**:
  - 活贝类:壳紧闭或轻触后立即闭合,拿起有重量感(说明有肉)
  - 贝肉:肉质饱满,颜色鲜亮(蛤蜊肉白色、扇贝肉橙黄色)
  - 气味:海鲜特有的鲜味,无臭味
- [ ] **贝类不新鲜特征**:外壳张开且触碰不闭合(死贝)、贝肉萎缩、有臭味、拿起很轻(空壳)

**3. 蔬菜类**:
- [ ] **叶菜类新鲜标准**(生菜、菠菜、油麦菜等):
  - 叶片:翠绿鲜亮,无黄叶、无枯萎
  - 质感:叶片脆嫩,手捏有水分感,不发软不发蔫
  - 茎部:茎部水嫩,掰断有清脆声,断面有水分渗出
  - 无病虫害:无虫眼、无霉斑、无腐烂
- [ ] **叶菜类不新鲜特征**:叶片发黄、枯萎、发软、有黏液、有异味、有腐烂斑点
- [ ] **根茎类新鲜标准**(土豆、萝卜、莲藕等):
  - 外表:表面光滑,无皱纹,无发芽,无霉斑
  - 质感:硬实饱满,手握有重量感,不发软不空心
  - 切面:切开后颜色正常(土豆白色、萝卜白色、莲藕白色),无黑斑,无纤维化
  - 气味:有食材特有清香,无霉味或腐臭味
- [ ] **根茎类不新鲜特征**:发芽、表面皱缩、发软、有霉斑、切开有黑斑或空心、有异味

**4. 豆制品类(豆腐、豆皮、豆干等)**:
- [ ] **新鲜标准**:
  - 色泽:豆腐洁白或淡黄色,豆皮淡黄色,无杂色斑点
  - 质感:豆腐有弹性,用手轻压不碎,表面光滑细腻
  - 气味:有豆类特有清香,无酸味、馊味
  - 保存液:豆腐浸泡水清澈透明,无浑浊,无异味
- [ ] **不新鲜特征**:
  - 色泽:发黄、发灰、有黏液、有霉斑
  - 质感:发黏、变软、变散、易碎
  - 气味:有酸味、馊味、臭味
  - 保存液:水变浑浊、有异味、有泡沫

**5. 蛋类**:
- [ ] **新鲜标准**:
  - 外观:蛋壳完整,表面粗糙有颗粒感,无裂纹
  - 手感:拿起有重量感,晃动时内部无明显晃动感(说明蛋白稠厚)
  - 透光检查:对着光源看,气室小(直径<6mm),蛋黄轮廓模糊(说明蛋白稠厚)
  - 打开后:蛋黄饱满凸起居中,蛋白浓稠分层明显(内层浓稠外层稀薄),无异味
- [ ] **不新鲜特征**:
  - 外观:蛋壳有裂纹、表面光滑无颗粒感
  - 手感:晃动时内部有明显晃动感或水声(蛋白稀薄)
  - 透光检查:气室大(>9mm),蛋黄轮廓清晰(蛋白稀薄)
  - 打开后:蛋黄扁平散开、蛋白稀薄无分层、有异味(硫化氢味、臭味)
  - 沉水测试:新鲜蛋沉入水底,不新鲜蛋漂浮或半浮

**6. 禽类(鸡、鸭等)**:
- [ ] **新鲜标准**:
  - 皮肤:皮肤干爽光滑,毛囊凸起明显,颜色淡黄或白色,无黏液
  - 肉质:肉质紧实有弹性,按压后立即回弹,切面有湿润感
  - 气味:有禽类特有的淡淡腥味,无异味
  - 眼睛:眼球饱满(如是整只禽类)
- [ ] **不新鲜特征**:皮肤发黏有黏液、肉质松软无弹性、有酸臭味或异味、皮肤发暗或有绿色斑点

**验收时必检项目**(每次食材到货必须全部检查)**:
- [ ] 1. 供应商资质:检查送货单上的供应商名称、联系方式、送货时间是否完整
- [ ] 2. 温度检查:使用食品温度计检查冷藏食材温度≤4℃、冷冻食材≤-18℃
- [ ] 3. 包装完整性:检查包装无破损、无开封、无污染
- [ ] 4. 标签信息:检查生产日期、保质期、储存条件标注清晰
- [ ] 5. 逐项新鲜度检查:按照上述分类标准,对每种食材执行"看闻摸查"
- [ ] 6. 不合格处理:发现不合格食材,当场拒收,拍照记录,在送货单上注明,要求供应商签字确认
- [ ] 7. 验收记录:填写《食材验收记录表》,记录日期、供应商、食材名称、数量、新鲜度判定结果、验收人签字

**生熟分离详细标准**:

**1. 刀具分离**:
- [ ] 生食刀:红色刀柄,用于切生肉、生海鲜、生蔬菜
- [ ] 熟食刀:蓝色刀柄,用于切熟食、熟肉、已烹饪食材
- [ ] 过敏原刀:黄色刀柄,专门用于处理过敏原食材(海鲜、坚果等)
- [ ] 刀具标识:刀柄上贴有岗位标签(如"切配-生"、"切配-熟"、"过敏原-海鲜")
- [ ] 严禁混用:每次使用前检查刀柄颜色,绝不可用红色刀切熟食
- [ ] **刀具存放位置**:
  - 生食刀架:放置在生食操作台右侧墙面,刀架贴红色标签"生食刀具"

**温度测量标准及使用方法**:

**1. 食品温度计类型与选择**:
- [ ] **探针式温度计**(推荐):
  - 量程:-50℃至300℃
  - 精度:±1℃
  - 探针长度:≥12cm(能插入食材中心)
  - 响应时间:≤5秒
  - 适用场景:测量固体食材(肉类、海鲜)、液体汤底、油温
- [ ] **红外温度计**(辅助):
  - 量程:-50℃至380℃
  - 测量距离:5-15cm
  - 适用场景:快速扫描冷柜表面温度、设备表面温度
  - 注意:不适合测量食材内部温度(只能测表面)

**2. 食材验收温度测量方法**:
- [ ] **冷藏食材测量**(目标温度0-4℃):
  - 测量位置:选择包装中最大块食材,将探针插入食材中心(深度≥5cm)
  - 等待时间:保持探针插入状态10-15秒,等待读数稳定
  - 多点测量:每批次至少测量3个不同位置,取平均值
  - 判定标准:所有测量点≤4℃为合格,任一点>4℃需重新评估
- [ ] **冷冻食材测量**(目标温度≤-18℃):
  - 测量位置:将探针插入食材间隙或包装边缘(冷冻食材太硬,避免强行插入损坏探针)
  - 等待时间:保持15-20秒,等待读数稳定
  - 判定标准:温度≤-18℃为合格,>-18℃需拒收
- [ ] **常温食材测量**(目标温度15-25℃):
  - 测量位置:放置在食材表面或插入包装内
  - 判定标准:温度15-25℃为正常,>30℃需评估是否变质

**3. 冷藏/冷冻设备温度监控**:
- [ ] **冰箱温度计读取方法**:
  - 读取时间:每日上午9:00、下午15:00、晚上21:00三次读取记录
  - 读取位置:站在冰箱正前方,视线与温度计刻度平行(避免仰视或俯视造成误差)
  - 记录方法:将读数记录在《温度监控记录表》,超标立即标红并报告
- [ ] **冷藏柜标准温度**:
  - 目标温度:0-4℃
  - 可接受范围:-2℃至6℃(短时波动)
  - 超标处理:温度>6℃或<-2℃超过30分钟,立即调整温控并报修
- [ ] **冷冻柜标准温度**:
  - 目标温度:-18℃至-20℃
  - 可接受范围:-15℃至-22℃
  - 超标处理:温度>-15℃或<-25℃,立即调整温控并报修

**4. 加工过程温度监控**:
- [ ] **食材加工中温度控制**:
  - 测量时机:食材取出后每30分钟测量一次
  - 测量方法:将探针插入食材中心,保持10秒读数
  - 标准要求:生鲜食材加工过程温度应≤4℃,超过4℃需立即放回冷藏柜
- [ ] **汤底温度测量**(烹饪中):
  - 测量位置:将探针插入汤底中心(距离锅底5-10cm)
  - 等待时间:保持10秒,确保读数稳定
  - 沸腾判定:温度≥98℃视为沸腾(高原地区约95℃)
  - 保温标准:汤底保温温度应≥75℃,低于75℃需重新加热
- [ ] **油温测量**(炸制时):
  - 测量方法:将探针斜插入油锅中心(探头完全浸入油中,但不触碰锅底)
  - 等待时间:保持5秒快速读数
  - 安全注意:手持探针尾部,避免接触热油
  - 常用温度标准:
    - 低温炸(120-140℃):炸花生、坚果
    - 中温炸(160-180℃):炸丸子、炸鱼
    - 高温炸(180-200℃):炸薯条、复炸使食物酥脆

**5. 温度计校准与维护**:
- [ ] **每日校准**(冰水法)**:
  - 准备冰水混合物:杯中加入冰块+纯净水,搅拌均匀
  - 将探针完全浸入冰水中(不触碰杯底和杯壁)
  - 等待30秒,读数应为0℃(±1℃为正常)
  - 如偏差>2℃,使用调节螺丝校准或更换温度计
- [ ] **每周校准**(沸水法)**:
  - 烧开一壶纯净水
  - 将探针浸入沸水中(不触碰容器)
  - 读数应为100℃(海拔每升高300米,沸点降低约1℃)
  - 如偏差>2℃,需校准或更换
- [ ] **日常维护**:
  - 每次使用后:用75%酒精棉片擦拭探针和外壳,自然晾干
  - 避免摔落:探针精密仪器,摔落会影响准确度
  - 定期更换电池:显示变暗或反应变慢时立即更换电池
  - 更换周期:温度计使用寿命约1-2年,建议每年更换一次确保精度

**6. 温度记录与异常处理**:
- [ ] **温度记录要求**:
  - 记录表格:使用《温度监控记录表》
  - 记录内容:日期、时间、测量位置(如"1号冷藏柜"、"牛肉盒")、温度读数、测量人签字
  - 记录频率:冷柜温度每日3次,食材加工每30分钟一次,验收每批次必测
- [ ] **温度异常处理**:
  - 发现超标:立即停止使用该批次食材,隔离并贴上"温度异常-待处理"标签
  - 评估原因:检查是否设备故障、食材放置时间过长、还是测量误差
  - 处理方案:
    - 轻微超标(冷藏5-7℃):立即冷藏,1小时内使用完毕
    - 严重超标(冷藏>7℃):丢弃处理,不得使用
  - 记录留存:在《温度异常处理记录表》详细记录,拍照存档

**过敏原管理制度**:

**1. 常见过敏原识别**:
- [ ] **海鲜类**:虾、蟹、鱼、贝类、鱿鱼、章鱼、鲍鱼等所有海产品
- [ ] **坚果类**:花生、腰果、杏仁、核桃、开心果等
- [ ] **蛋类**:鸡蛋、鸭蛋、鹅蛋及蛋制品
- [ ] **乳制品**:牛奶、奶酪、黄油、奶油、炼乳等含奶制品
- [ ] **麦类**:小麦、大麦、黑麦等含麸质谷物及其制品(面粉、面条、面包等)
- [ ] **豆类**:黄豆、豆腐、豆浆、豆瓣酱等大豆制品
- [ ] **芝麻**:芝麻、芝麻油、芝麻酱等芝麻制品
- [ ] **其他**:芹菜、芥末、木姜子等特殊香料

**2. 过敏原储存管理**:
- [ ] **专区存放**:过敏原食材必须在冷藏/冷冻柜中设立专门区域
- [ ] **明确标识**:
  - 在储存柜外贴黄色警示标签"过敏原食材专区"
  - 每种过敏原食材容器上贴标签,标注"过敏原-海鲜"、"过敏原-坚果"等
  - 标签位置:容器正面中央,字体≥3cm高,清晰可见
- [ ] **隔离存放**:过敏原食材与普通食材之间用隔板或专用容器隔开,避免接触
- [ ] **分层存放**:易引发严重过敏的食材(海鲜、坚果)放置在下层,减少污染风险

**3. 过敏原加工管理**:
- [ ] **专用工具**:
  - 过敏原食材使用黄色刀柄刀具、黄色砧板
  - 工具上贴"过敏原专用"标签(如"过敏原-海鲜刀")
  - 严禁与其他食材共用刀具、砧板、容器
- [ ] **专区操作**:
  - 在操作台设立"过敏原加工专区",贴黄色标识线
  - 该区域只能处理过敏原食材,不可处理其他食材
  - 专区与普通操作区距离≥1米
- [ ] **操作顺序**:
  - 先处理不含过敏原的食材,最后处理过敏原食材
  - 或处理完过敏原食材后,必须完成以下流程才能处理其他食材:
    1. 脱掉手套,丢入垃圾桶
    2. 用七步洗手法洗手,洗手时间≥30秒
    3. 更换干净工作服(如过敏原溅到衣服上)
    4. 清洁消毒操作台面
    5. 更换新手套
- [ ] **工具清洁**:
  - 过敏原工具使用后立即清洗,用流动水冲洗≥1分钟
  - 用洗涤剂刷洗,去除所有食材残留
  - 用84消毒液(1:100)浸泡消毒30分钟
  - 冲洗干净,沥干后放入专用工具柜

**4. 过敏原交叉污染预防**:
- [ ] **避免飞溅**:切配过敏原食材时动作轻缓,避免汁液飞溅到其他食材或台面
- [ ] **及时清理**:过敏原食材掉落台面立即清理,用消毒液擦拭该区域
- [ ] **容器加盖**:处理过敏原食材时,已完成的其他食材必须加盖保存
- [ ] **手套更换**:接触过敏原后必须更换手套,不可戴同一副手套接触其他食材
- [ ] **工作服清洁**:过敏原溅到工作服上,必须立即更换,不可穿着污染工作服继续操作

**5. 客人沟通与记录**:
- [ ] **主动询问**:前厅服务员在客人点单时主动询问"您是否对任何食材过敏?"
- [ ] **信息传递**:
  - 服务员在点单系统备注栏标注"过敏-海鲜"或"过敏-坚果"
  - 厨房小票上打印显示过敏信息
  - 传菜员口头向厨师长确认过敏信息
- [ ] **菜品调整**:
  - 厨师长根据过敏信息调整菜品,去除过敏原食材
  - 使用专门容器制作,避免交叉污染
  - 在出品时用特殊颜色标签(如黄色)标注"无海鲜"
- [ ] **记录留存**:在《过敏原客人记录表》填写日期、桌号、过敏类型、调整方案,保存备查

**6. 应急处理**:
- [ ] **发现过敏反应**:客人出现皮肤红肿、呼吸困难、呕吐等过敏症状,立即通知店长/经理
- [ ] **停止用餐**:立即停止客人食用该菜品,收回菜品封存
- [ ] **紧急救助**:
  - 轻微过敏:提供清水漱口,询问是否需要就医
  - 严重过敏:立即拨打120,同时让客人平躺,保持呼吸通畅
- [ ] **事件记录**:详细记录时间、菜品、症状、处理过程,保存菜品样品和小票
- [ ] **后续跟进**:店长联系客人了解恢复情况,根据公司制度处理

#### 餐具卫生
- [ ] 餐具清洗消毒后统一存放,不能直接放在台面,防止污染
- [ ] 用过的餐具经过初洗、清洁剂清洗、清水清洗、消毒四道工序
- [ ] 餐具内外干净干燥,无油迹、无清洁剂泡
- [ ] 所有清洗消毒后餐具统一盛放在柜中
- [ ] 用餐前餐具集中整齐摆放,保持清洁,用洁净白布盖好,防止蝇叮
- [ ] 餐具未经消毒不得循环使用

#### 餐具消毒方法
- [ ] **煮沸、蒸汽消毒**:消毒温度100℃并保持10分钟以上
- [ ] **远红外消毒柜消毒**:柜内温度达到120℃,并保持10分钟以上
- [ ] **洗碗机消毒**:水温达到85℃,餐具冲洗消毒时间大于40秒
- [ ] **化学药物消毒(84消毒液)**:
  - 比例: 1:200 (即1份消毒液+199份水)
  - 具体操作: 10ml消毒液 + 1990ml水 = 2000ml溶液
  - 或简便方法: 1瓶盖(约5ml)消毒液 + 1瓶(1000ml)矿泉水
  - 有效氯浓度: 约250-300mg/L
  - 浸泡时间: 5分钟以上
  - 将餐具全部浸入液体中作用5分钟以上,用流动的净水冲去餐具表面的残液

1. [ ] 颜色不正不出(看)
2. [ ] 味道不佳不出(尝)
3. [ ] 标准不够不出(秤)
4. [ ] 摆盘不美不出(查)
5. [ ] 原料变质不出(闻)
6. [ ] 温度不够不出(量)
7. [ ] 餐具不洁不出(擦)

### 1.5 进销存管理

#### 盘点标准
- [ ] **盘点时间**:每月月底最后一天
- [ ] **盘点范围**:库房、厅面、吧台、厨房
- [ ] **盘点的物品**:低值易耗、固定资产、物料、餐具、酒水(饮品)
- [ ] **盘点标准**:以财务每月系统导出盘点表为标准
- [ ] **盘点的方法**:按固定路线分区盘点,从上方货架、中间台面、下方冰箱顺序逐一盘点

#### 先进先出规范
- [ ] 生产日期在前的优先出库;入库时间长的优先出库
- [ ] 所有入库物料相关标签齐备;批号/日期填写清楚规范准确
- [ ] 先到的物料先使用,后到的物料后使用
- [ ] 物料存放遵照先里后外、先下后上;取用时按相同顺序
- [ ] 成品、半成品、开封后的原料,加膜加盖存放于指定位置,填写效期卡
- [ ] 定期检查库存/效期,确保先进先出原则正确执行,确保库存中没有过期或临近过期产品

#### 食材验收标准
- [ ] **数量核对**: 与送货单逐一核对,确保数量准确无误
- [ ] **质量检查**: 检查食材颜色、气味、新鲜度是否正常
- [ ] **效期检查**: 查看生产日期和保质期,确保有充足使用时间
- [ ] **包装完整性**: 检查包装有无破损、污染、漏气、渗漏等现象
- [ ] **称重确认**: 对大宗食材进行抽查称重,核实净重
- [ ] **温度检查**: 冷藏/冷冻食材验收时检查温度是否符合要求
- [ ] **不合格品处理**: 当场拒收或拍照记录后联系供应商退换货,做好登记

#### 食材验收异常处理流程

**1. 数量不符异常**:
- [ ] **发现问题**: 实际数量与送货单不符(多送或少送)
- [ ] **立即处理**:
  - 停止验收,保持食材原包装不动
  - 当场与送货员核对送货单和实物
  - 拍照记录实物数量(拍清食材+送货单)
- [ ] **处理方案**:
  - 少送:在送货单上注明"实收XX,少送XX",要求送货员签字确认,联系供应商当日补送或次日补足
  - 多送:在送货单上注明"实收XX,多送XX",拍照后退回多余部分,或收下并在系统中标注需扣款
- [ ] **记录留存**: 填写《验收异常记录表》,上传照片至供应商管理系统,抄送店长

**2. 质量不合格异常**:
- [ ] **发现问题**: 食材出现变质、异味、变色、霉变、破损等质量问题
- [ ] **立即处理**:
  - 立即停止验收,将问题食材单独隔离
  - 拍照记录问题细节(多角度拍摄:整体+特写+标签+送货单)
  - 使用"看闻摸查"标准确认具体问题(参考1.4食品卫生-食材新鲜度检查标准)
- [ ] **处理方案**:
  - 当场拒收:在送货单上注明"质量不合格-拒收",列明具体问题(如"牛肉发黑有异味"),要求送货员签字,当场退回
  - 联系供应商:立即致电供应商说明情况,要求当日重新配送合格品
  - 应急采购:如该食材为当日必需品且供应商无法及时补送,立即启动应急采购(就近市场采购或联系备用供应商)
- [ ] **记录留存**: 填写《验收异常记录表》,详细描述问题,上传照片,供应商确认后归档

**3. 效期异常**:
- [ ] **发现问题**: 食材已过期、临期(剩余保质期<20%)、或无标签标识
- [ ] **立即处理**:
  - 停止验收,将该批次食材单独放置
  - 拍照记录标签(生产日期、保质期、批号)
  - 计算剩余保质期百分比
- [ ] **处理方案**:
  - 已过期:直接拒收,在送货单上注明"已过期-拒收",要求送货员签字确认并退回
  - 临期品(剩余<20%):拒收,要求供应商更换新鲜批次
  - 无标签:拒收,要求供应商提供带完整标签的合格产品
- [ ] **记录留存**: 填写《验收异常记录表》,记录批号、日期、处理结果

**4. 包装破损异常**:
- [ ] **发现问题**: 外包装破损、漏气、污染、封口开裂、真空袋漏气等
- [ ] **立即处理**:
  - 不打开破损包装,保持原状
  - 拍照记录破损细节(多角度拍摄)
  - 检查内容物是否受污染
- [ ] **处理方案**:
  - 外包装轻微破损但内包装完好:与送货员协商,降价收货或拒收
  - 内包装破损或内容物污染:直接拒收,在送货单上注明"包装破损-拒收",退回
  - 冷冻品解冻:直接拒收(包装破损导致解冻视为质量不合格)
- [ ] **记录留存**: 拍照记录,填写异常单,供应商确认签字

**5. 温度不达标异常**:
- [ ] **发现问题**: 冷藏食材温度>4℃,冷冻食材温度>-18℃
- [ ] **立即处理**:
  - 使用食品温度计测量食材中心温度
  - 拍照记录温度计读数+食材包装
  - 检查送货车辆冷链设备是否正常
- [ ] **处理方案**:
  - 温度轻微超标(冷藏4-7℃,冷冻-15至-18℃):评估食材状态,如无异味、变质,可收货但标注"需优先使用"
  - 温度严重超标(冷藏>7℃,冷冻>-15℃):直接拒收,在送货单上注明"温度不达标-拒收"
  - 送货车无冷链设备:直接拒收全部冷藏冷冻食材,联系供应商整改
- [ ] **记录留存**: 记录温度数据,拍照留证,填写异常单

**6. 供应商资质异常**:
- [ ] **发现问题**: 送货员无健康证、送货单无公司印章、车辆无冷链设备等
- [ ] **立即处理**:
  - 要求送货员出示健康证、送货车辆检验证明
  - 检查送货单是否有供应商公章、日期、联系方式
- [ ] **处理方案**:
  - 资质不全:拒收全部食材,要求供应商补齐资质后重新配送
  - 紧急情况:收货但标注"资质待补",要求供应商当日补交资质证明,否则次日起停止合作
- [ ] **记录留存**: 拍照记录,上报店长,填写异常单

**7. 紧急应急采购流程**(供应商无法及时补货时):
- [ ] **评估需求**: 确认缺货食材是否为当日必需(查看当日预定单、历史销售数据)
- [ ] **启动应急**:
  - 店长/厨师长授权应急采购
  - 选择采购渠道:①就近农贸市场 ②备用供应商 ③其他门店调货
- [ ] **采购标准**:
  - 采购人员携带《食材验收记录表》和食品温度计
  - 按照正常验收标准选购(执行"看闻摸查"标准)
  - 索要销售凭证(小票或发票)
  - 拍照记录采购过程(摊位/商家+食材+凭证)
- [ ] **费用报销**: 当日填写《应急采购申请表》,附上凭证和照片,店长审批后报销
- [ ] **后续跟进**: 第二天向原供应商说明情况,要求补偿差价或给予折扣

#### 盘点参与要求
- [ ] **盘点参与**: 所有岗位均需参与月度盘点,负责本岗位区域的物品清点
- [ ] **盘点时间**: 每月月底最后一天进行全面盘点
- [ ] **新店强化盘点**: 新店期(开业前2个月)要求每周盘点一次,确保进销存管理规范

### 1.6 消防安全管理

#### 灭火器使用标准
- [ ] 保存在干燥通风且方便取用的地方,避免受潮、爆晒
- [ ] 定期检查各个部件是否完整,是否影响正常使用
- [ ] 检查是否达到规定的维修年限
- [ ] 对灭火器定期保养/维护/清洁
- [ ] 周围保持干净,禁止堆放杂物

#### 灭火器检查要点
- [ ] 检查压力表外表面无变形、损伤,指针处在绿区
- [ ] 喷嘴无变形、开裂及损伤
- [ ] 喷射软管畅通,无变形和损伤
- [ ] 压把、阀体等金属件无严重损伤、变形、锈蚀
- [ ] 保险销和铅封完好,未被开启喷射
- [ ] 筒体严重变形和锈蚀必须报废

#### 消防日常管理
- [ ] 安全出口通道不堆放任何物品
- [ ] 消防喷淋脱落不要用封箱带粘或胶水固定
- [ ] 定时查看各区域灭火器保质期,快过期及时上报更换
- [ ] 灭火器箱损坏无法维修要上报更换
- [ ] 应急电源及安全指示牌每周放一次电,检查设备运转是否正常
- [ ] 定期检查消火栓是否完好,有无生锈现象
- [ ] 检查消火栓供水闸阀是否渗漏水
- [ ] 消火栓定期放水检查
- [ ] 检查报警按钮、指示灯及控制线路

### 1.10 标准流程模板

> **说明**: 为避免重复,以下模板适用于所有岗位。各岗位章节只列出特定要求,通用部分参照此模板执行。
>
> **术语说明**:
> - **SOP**: Standard Operating Procedure,标准作业程序,指经过优化后的操作流程和标准
> - **4D管理**: 整理到位、责任到位、培训到位、执行到位的现场管理方法
> - **先进先出**: First In First Out(FIFO),先到的物料先使用的库存管理原则

#### 开市流程通用标准

**仪容仪表检查**
- [ ] 参照[仪容仪表标准](#11-仪容仪表标准)

**班前会**
- [ ] 集合点名,清楚部门人员当日配置情况、到岗情况
- [ ] 领班根据集合传达的工作安排及任务要求做好工作拆分
- [ ] 协助配合其他岗位的餐前准备工作

**开市环境及安全检查**
- [ ] 检查收市卫生工作是否按标准要求完成,不符合的拍照反馈并整改
- [ ] 检查电路、网络、电源插座是否正常无安全隐患
- [ ] 检查设备是否正常运转
- [ ] 检查冰箱内食品是否生熟分离,温度是否正常
- [ ] 检查前一天剩余食材是否变质变色变味
- [ ] 检查是否按收市标准封膜加盖、分类分区、分层收纳储存
- [ ] 原料效期检查,是否变质/异味
- [ ] 检查原料/辅料是否充足
- [ ] 设备出现故障立即上报(不超过5分钟),2小时内解决或启动应急预案;短时间内不能解决的,采用应急方案并向店总报备

**原料储存标准**
- [ ] 主辅料必须遵循先进先出原则
- [ ] 进入冰箱食材必须:分类、分区、分层、分颜色

**用品用具收纳标准**

- [ ] 设施设备卫生(参照设施设备卫生标准)

#### 营业中服务流程通用标准

- [ ] 清楚管理组负责人
- [ ] 对讲机开启并正常使用
- [ ] 严格按照产品SOP标准执行
- [ ] 查看食材/物料数量是否满足营业用量,不足时及时预制
- [ ] 查看辅料是否充足,根据当天预估量进行补充

- [ ] 日常物料卫生
- [ ] 设备检查

- [ ] 电路、电源检查
- [ ] 四害检查

## 1.11 食材申购统一流程

> **说明**: 明确各岗位食材申购责任与流程,确保次日食材供应充足

### 申购责任划分
- [ ] **各岗位负责本档口食材申购**:小吃房、热菜岗、荤菜/素菜岗、凉菜岗等各岗位根据库存和预估销量申购本档口所需食材
- [ ] **厨师长负责汇总与下单**:统一收集各岗位申购单,汇总后向供应商下单
- [ ] **店长负责审核与监督**:对申购数量和金额进行审核,避免过度采购或库存积压

### 申购流程
1. **各岗位填写申购单** (每日14:30前完成):
   - [ ] 根据当日实际销售情况统计库存
   - [ ] 根据次日预估销量计算需求
   - [ ] 填写《食材申购表》,注明品名、规格、数量、单位
   - [ ] 14:30前将申购单提交给厨师长

2. **厨师长汇总处理** (每日14:30-15:30):
   - [ ] 收集各岗位申购单
   - [ ] 核对申购数量是否合理,必要时调整
   - [ ] 汇总整理,统一向各供应商下单
   - [ ] 将采购清单公示在厨房白板,便于次日验收

3. **供应商送货** (次日8:00-8:30或14:00-15:00):
   - [ ] 供应商按照采购清单配货
   - [ ] 次日8:00-8:30或14:00-15:00送货到店,避开班前会和餐中高峰期
   - [ ] 厨师长+店长双签验收

### 申购原则
- [ ] **根据历史销量**:参考近7日同类日期(如周一参考上周一)的销售数据
- [ ] **根据预估销量**:考虑天气、节假日、促销活动等因素调整预估
- [ ] **根据库存余量**:盘点当日库存,计算次日实际需求=预估销量-库存余量
- [ ] **避免过度采购**:新鲜食材尽量当日采购当日使用,减少隔夜储存

### 兜底机制
- [ ] **厨师长补充基础食材**:如某岗位忘记申购,厨师长根据历史数据补充基础食材(如米、油、盐等常用品)
- [ ] **特殊食材缺货责任**:特殊食材(如进口调料、稀有食材)缺货由忘记申购的岗位负责
- [ ] **提醒改进**:厨师长在班前会提醒改进,避免重复发生

## 1.12 关键工作原则

### 食材管理原则
- [ ] **先进先出原则(FIFO)**:优先使用日期较早的食材,避免过期浪费
- [ ] **生熟分开存放**:熟食上层,生食下层,避免交叉污染
- [ ] **加膜加盖密封保存**:所有食材必须加保鲜膜+加盖,防止污染和串味
- [ ] **定期检查食材品质和效期**:每日餐后检查,发现变质立即处理

### 出品质量原则
- [ ] **按SOP标准执行**:严格按照标准操作程序制作,确保品质一致
- [ ] **按下单顺序出品**:遵循先下单先出餐的原则,公平对待每位顾客
- [ ] **确保出品时间符合规定**:控制出餐时间,避免顾客长时间等待
- [ ] **特殊需求必须明确传达**:对于顾客的特殊要求(如不加辣、过敏原),必须明确标注并传达

### 卫生安全原则
- [ ] **保持个人卫生**:参照[仪容仪表标准](#11-仪容仪表标准)
- [ ] **维护环境卫生**:随时保持工作区域清洁,一餐一清
- [ ] **工具消毒到位**:刀具、砧板、餐具按规定消毒,参照[消毒标准](#14-食品安全标准)
- [ ] **防止交叉污染**:生熟分离、过敏原标识、分色管理,参照[4D现场管理标准](#13-4d现场管理标准)

### 2.1.5 检查与监督机制

> **说明**: 日常检查与定期抽查结合,确保标准执行到位

#### 每日必查(快速巡视)
- [ ] **10:40-11:00 开餐前各档口备餐情况**:
  - 目视检查,发现问题当场指出
  - 重点检查食材备货是否充足、设备是否正常
  - 用时15-20分钟

- [ ] **13:30 午餐后卫生清洁情况**:
  - 走一圈,重点看台面地面
  - 检查工具是否清洗归位
  - 用时5-10分钟

- [ ] **21:30-21:45 闭店安全检查**:
  - 三关一闭(关水、关电、关燃气、闭门窗)
  - 重点检查燃气阀门、电器设备是否关闭
  - 用时10-15分钟

#### 每周抽查(每周2-3次,随机)
- [ ] **冰箱食材存放检查**:
  - FIFO原则执行情况
  - 生熟分开、加膜加盖情况
  - 食材效期和新鲜度

- [ ] **工具清洁消毒检查**:
  - 刀具、砧板、餐具清洁情况
  - 消毒记录是否填写
  - 工具摆放是否规范

- [ ] **设备维护保养检查**:
  - 按[厨房设备归属表](#36-厨房设备归属表)检查
  - 设备是否正常运转
  - 日常维护是否到位

#### 每月深度检查(月末最后一天)
- [ ] **所有设备设施全面检查**:
  - 大型设备(灶台、冰箱、油烟机等)
  - 小型设备(刨片机、炸缸、搅拌机等)
  - 记录需要维修或更换的设备

- [ ] **库存盘点与效期检查**:
  - 配合月度盘点,参照[进销存管理](#15-进销存管理)
  - 重点检查临期食材和库存积压
  - 提出采购和库存优化建议

- [ ] **卫生死角深度清理**:
  - 墙角、下水沟、设备背面等平时难以清洁的区域
  - 组织全体厨房人员集中清理
  - 确保通过卫生检查

#### 检查原则
- [ ] **日常检查重点看关键节点**,不必事事签字,避免流于形式
- [ ] **发现问题先口头提醒**,重复问题才书面警告,给予改正机会
- [ ] **抽查结果在班前会上通报**(不点名,只说问题),保护员工自尊
- [ ] **鼓励做得好的岗位**,给予正向激励,营造积极氛围

- [ ] 用过的餐具要经过四道工序处理:初洗(冲掉残渣)→清洁剂清洗(去油污)→清水清洗(冲净洗洁精)→消毒(84消毒液按1:200配制或高温消毒,具体配比参见[餐具消毒方法](#14-食品安全标准)),餐具内外要干净干燥,无油迹、无清洁剂泡
- [ ] 盛放消毒餐具的保洁柜要有明显标记,要经常擦洗消毒,已消毒和未消毒的餐饮具要分开存放
- [ ] 洗刷餐饮具的水池专用,不得在洗餐饮具池内清洗食品原料,不得在洗餐饮具池内冲洗拖布等
- [ ] 操作时注意保持卫生,洗刷消毒结束,要清理地面、水池卫生,每30分钟清理一次泔水桶,做到无油渍残渣,泔水桶内外清洁

> **注**: 其他通用要求参照[4D现场管理标准](#13-4d现场管理标准)

### 2.4.1 P0级关键风险点(A4-01至A4-10)

#### A4-01 锅底配方标准化(最高优先级)

**风险描述**
- 问题: 火锅锅底味道不一致,导致顾客投诉率高
- 根因: 厨师按照习惯配方而非标准配方操作

**防呆执行标准**
- [ ] 建立《锅底配方标准卡》贴于炉灶旁,写清楚:
  - 配方名称(如"麻辣锅底""番茄锅底"等)
  - 主要配料及克重(精确到50克以内)
  - 配比比例(如"1:1:1:2")
  - 烹饪步骤及时间节点
  - 成品特征(颜色、香气、浓度等)
- [ ] **强制执行点**: 每次配锅底前，热菜厨师必须指读标准卡一遍，厨师长每班抽查不少于2次配锅
- [ ] **检查方法**: 配锅后10分钟内品尝，核对4项指标:
  1. 颜色深度: 与标准色卡对比(拍照对比)
  2. 香气: 与标准香气描述对比(如"浓郁麻香"或"微辣番茄香")
  3. 口感: 咸淡度测试(用测盐度仪测)
  4. 浓度: 观察油脂分布
- [ ] **不合格处理**: 若不符合标准,即刻倒出重配,不可继续使用

**关键要素**
| 配方名称 | 主料1 | 主料2 | 调料 | 火候 | 时长 | 成品特征 |
|---------|-------|-------|------|------|------|----------|
| 麻辣锅底 | 牛油500g | 豆瓣200g | 花椒100g | 中火 | 15min | 深红色,麻香浓郁 |
| 番茄锅底 | 番茄酱300g | 高汤800ml | 盐15g | 小火 | 10min | 鲜红色,酸甜适中 |
| 清汤锅底 | 高汤1000ml | 冰糖10g | 盐8g | 小火 | 5min | 清澈透亮,鲜香 |

**责任人及检查点**
- **执行者**: 热菜厨师(配锅)
- **验收者**: 厨师长(10分钟内品尝检查)
- **签字**: 配锅记录表上签字确认

#### A4-02 酸汤pH值监控(食品安全关键)

**风险描述**
- 问题: 酸汤pH值过高(>4.5)或过低(<3.5),影响食材保存和口感
- 根因: 未实时监测,凭经验调整

**防呆执行标准**
- [ ] **配置**: 每个炉灶配备食品pH值测试纸或pH计
- [ ] **标准范围**: 3.8-4.2 (最佳范围为4.0±0.2)
- [ ] **测量频率**:
  - 开市: 配锅完毕后立即测
  - 营业中: 每2小时测一次(11:00、13:00、17:00、19:00、21:00)
  - 补菜前: 每次加新配料前都要测
- [ ] **测量方法**:
  1. 用干净玻璃棒取样5ml酸汤
  2. 浸入pH测试纸或pH计探头
  3. 对比标准色卡或读取数值
  4. 记录时间+pH值于《酸汤监控日志表》
- [ ] **不合格处理**:
  - pH>4.5: 立刻加白醋(每次加5ml,加后等5分钟重新测)
  - pH<3.5: 立刻加高汤或清水(每次加10ml,加后等5分钟重新测)
  - 严禁继续使用，确认达标后才可继续营业

**关键工具**
- [ ] pH测试纸(价格低,需存放干燥处)或pH计(精准但需定期校准)
- [ ] 酸汤监控日志表(每班填写)
- [ ] 白醋和高汤(应急调整用)

**责任人及检查点**
- **执行者**: 热菜厨师(每2小时测一次)
- **监督者**: 厨师长(每班检查不少于1次测量记录)
- **签字**: 日志表由执行者填写并签字

---

#### A4-03 烹饪温度控制(品质与安全关键)

**风险描述**
- 问题: 菜品温度不足85°C或过高>100°C,影响口感和食材营养
- 根因: 用手感或目测判断,无量化标准

**防呆执行标准**
- [ ] **配置**: 每个灶台配备食品温度计(红外线或插式)
- [ ] **标准温度**:
  - 爆菜: 90-95°C (锅底温度)
  - 炖煮: 85-90°C (保持温度)
  - 热菜出品: ≥85°C (测量菜品中心温度)
- [ ] **测量频率**:
  - 开市: 锅底加热到目标温度,确认后开始营业
  - 营业中: 每道菜出品前测一次,记录温度
  - 菜品出品: 从锅到传菜员手中前,再测一次(防止途中温度下降)
- [ ] **测量位置**:
  - 爆菜: 锅底油/汤的温度(用红外温度计从上方指向锅底)
  - 炖煮: 汤汁中心位置
  - 出品菜: 菜品肉类或蔬菜的中心温度(插式温度计)
- [ ] **记录方式**: 于《每日烹饪温度记录表》上记录:
  - 时间 | 菜品名称 | 测量位置 | 温度(°C) | 是否合格 | 执行者签字

**不合格处理**
- 温度<85°C: 菜品不可出品,重新加热至标准温度
- 温度>100°C: 等待2分钟后再出品(防止顾客烫伤)

**责任人及检查点**
- **执行者**: 热菜/凉菜厨师(每道菜出品前测)
- **监督者**: 厨师长(每班抽查不少于3道菜的温度)
- **签字**: 记录表由执行者填写并签字

#### A4-04 品质四大检验标准(看/闻/尝/查)

**风险描述**
- 问题: 有的菜品带出菜品质量不稳定(颜色不对、味道变淡、口感不佳等)
- 根因: 出菜前没有进行系统的品质检验

**防呆执行标准**

每道菜出品前必须进行"4S品质检验"(看-闻-尝-查),任何一项不达标就不可出菜:

**1. 看(色泽检查)**
- [ ] 与菜品标准色卡对比(建议用手机拍照与标准图对比)
- [ ] 颜色是否正常(如红菜不应该是黄色、青菜不应该焦黄等)
- [ ] 视觉美感: 配菜搭配是否协调、盛盘是否规整
- 不合格: 若颜色异常,立刻查原因(火候不足/过度、调料错误等)并重做

**2. 闻(香气检查)**
- [ ] 是否有应有的香气(如炒菜应该有"镬气",红汤菜应该有"麻辣香")
- [ ] 是否有异味(焦味、腥味、馊味等)
- [ ] 香气浓度是否达标(不可太淡,更不可太冲)
- 不合格: 若有异味或香气淡,查原因(油不够、配料漏加、火候不足等)

**3. 尝(口感检查)**
- [ ] 咸淡度是否恰好(不可太咸也不可淡而无味)
- [ ] 其他口感(如麻感、酸感、甜感等是否均衡)
- [ ] 食材是否新鲜(无变质味道)
- 方法: 用干净调羹尝一口(测完立刻漱口)
- 不合格: 若口感不对,立刻调整(加盐/减盐/加调料等)

**4. 查(细节检查)**
- [ ] 菜品内是否混入杂物(头发、蛋壳、污染物等)
- [ ] 食材大小是否均匀(如土豆块太大或太小,肉片太厚或太薄)
- [ ] 是否有食材变质迹象(如肉类变色、蔬菜烂掉等)
- [ ] 是否有漏加配菜(如应该有香芹却没有等)
- 方法: 用眼睛仔细看,必要时用手轻拨检查
- 不合格: 若发现杂物或变质,不可出菜

**关键工具**
- [ ] 菜品标准色卡 (贴于出菜口)
- [ ] 品质检验表 (记录每道菜是否通过4项检验)
- [ ] 干净调羹 (用于尝菜)
- [ ] 手电筒 (用于查看细节)

**责任人及检查点**
- **执行者**: 热菜/凉菜/明档厨师(每道菜出品前执行)
- **监督者**: 厨师长(每班巡检不少于5次,检查执行情况)
- **签字**: 品质检验表由执行者填写并签字

---

#### A4-05 出餐速度达标控制(顾客体验关键)

**风险描述**
- 问题: 顾客等餐时间过长(>8分钟),导致投诉
- 根因: 出菜时间不稳定,无明确的时间标准和进度控制

**防呆执行标准**

**1. 菜品出菜时间标准**
- [ ] 爆菜(干炒、快速烹饪): 点单后3-5分钟出品
- [ ] 炖煮(汤类、长时间烹饪): 点单后5-8分钟出品
- [ ] 凉菜(拌、冷制): 点单后2-3分钟出品
- [ ] 明档(现成展示): 点单后1分钟出品

**2. 进度管理(严格按时间节点执行)**

| 时间节点 | 工作内容 | 执行者 | 检查点 |
|---------|---------|--------|---------|
| T+0秒 | 热菜厨师收单,开始烹饪 | 热菜厨师 | 小票机单据清晰可读 |
| T+1分钟 | 锅底烧热,加入主料 | 热菜厨师 | 火势均匀,不呛火 |
| T+2分钟 | 加入配料/调料 | 热菜厨师 | 调料无漏加 |
| T+3分钟 | 试菜(4S检验),准备出菜 | 热菜厨师+厨师长 | 品质检验表已填 |
| T+3-5分钟 | 装盘,加配菜和装饰物 | 明档厨师 | 盛盘规范,配菜齐全 |
| T+5分钟 | 最后检查(温度/颜色/配菜),调用传菜 | 厨师长 | 温度≥85°C,外观达标 |
| T+5-8分钟 | 传菜员传菜给顾客 | 传菜员 | 无中途掉落/遗漏 |

**3. 超时处理机制**
- [ ] 若T+5分钟还未出品(除特殊菜品外),立刻启动"加速模式":
  - 厨师长亲自跟进,检查卡在哪个环节
  - 若是品质问题,加快品质检验(不降低标准)
  - 若是烹饪问题,加火或调整火候
  - 若是装盘问题,明档加快速度
- [ ] 若T+8分钟还未出品,立刻报告厨师长:
  - 是否要暂时出品(品质可能受影响)
  - 是否通知顾客(告知预计等待时间)
  - 是否启动应急方案(如菜品不可做,提议替换菜品)

**4. 出菜速度统计与改进**
- [ ] 每班统计出菜速度达标率:
  - 公式: 达标菜品数 / 总菜品数 × 100%
  - 目标: ≥95%
  - 记录于《每日出菜速度统计表》
- [ ] 每周复盘:
  - 若达标率<95%,分析原因(人员不足/设备故障/特殊菜品多等)
  - 制定改进方案(如增加人手/调整菜单/提前准备等)

**关键工具**
- [ ] 出菜时间标准表 (贴于热菜台)
- [ ] 计时器或秒表 (用于计时)
- [ ] 小票机 (记录点单时间)
- [ ] 每日出菜速度统计表 (记录每班统计数据)

**责任人及检查点**
- **执行者**: 热菜/凉菜/明档厨师(按规定时间烹饪)
- **监督者**: 厨师长(实时监测,>3分钟无进展时介入)
- **签字**: 统计表由厨师长填写并签字

#### A4-06 食材新鲜度把控(食品安全关键)

**风险描述**
- 问题: 使用了变质或不新鲜的食材,导致食品安全事故或顾客投诉
- 根因: 出菜前没有逐一检查食材新鲜度

**防呆执行标准**

**1. 食材新鲜度检查标准(看-闻-摸)**

| 食材类型 | 新鲜标准 | 变质信号 |
|---------|---------|----------|
| 牛肉/羊肉 | 颜色鲜红,有光泽,无异味,摸起来有弹性 | 变黑/褐色,有腐败臭味,摸起来发软无弹性 |
| 豆制品 | 白色或淡黄,无异味,无发硬或软化 | 发黄/发黑,有酸味/馊味,发硬或发软 |
| 蔬菜 | 颜色鲜艳,无褐色斑点,挺立有水分 | 蔫蔫的,有黑斑,发软如水 |
| 鱼类/虾 | 眼睛清亮,鳃呈鲜红,无腥臭味 | 眼睛浑浊,鳃呈黑色,强烈腥臭味 |
| 豆芽 | 白色挺立,根系洁白,清香味 | 发黑软化,根系变棕,有酸臭味 |
| 冻品 | 表面无冰霜,无破损,无异味 | 有厚冰霜,包装破损,有异味 |

**2. 执行流程**
- [ ] **出菜前检查** (必须执行,无例外):
  1. 取出该菜品所需食材
  2. 依次进行"看-闻-摸"检查
  3. 若任何一项不符合新鲜标准,立刻停止使用,报告厨师长
  4. 在《食材新鲜度检查表》上记录检查结果(日期/时间/食材/结果/签字)

- [ ] **不合格食材处理**:
  1. 立刻隔离该食材,不混入其他食材
  2. 报告厨师长,确认是否可换菜品
  3. 如无替代菜品,立刻通知前厅,告知顾客该菜品缺货
  4. 将不合格食材记录于《食材不合格处理单》,并向供应商反馈

**3. 食材储存规范(防止变质)**
- [ ] 冷藏食材存放温度: 0-4°C (每日检查冰箱温度)
- [ ] 冷冻食材存放温度: -18°C以下
- [ ] 储存时间限制:
  - 冷藏肉类: ≤3天
  - 冷藏蔬菜: ≤5天
  - 冷藏豆制品: ≤2天
  - 冷冻肉类: ≤30天
- [ ] 生熟分开存放(避免交叉污染)
- [ ] 加膜加盖存放,明确效期卡(写清楚"入库日期"和"使用期限")

**责任人及检查点**
- **执行者**: 热菜/凉菜/明档厨师(出菜前逐一检查)
- **监督者**: 厨师长(每班巡检不少于2次,检查冰箱温度和食材状态)
- **签字**: 检查表由执行者填写并签字

#### A4-07 调料比例标准化

**风险描述**
- 问题: 调料加得多或少,导致菜品口味不一致
- 根因: 凭经验加调料,无定量标准

**防呆执行标准**
- [ ] 建立《调料配比标准表》:
  - 列出每道菜的主要调料(如盐、糖、酱油、醋等)
  - 精确到克或毫升(如"盐5克""酱油10ml")
  - 用量勺或量筒配合使用(不可用手抓或眼看)
- [ ] **强制执行点**: 每次加调料前,热菜厨师指读标准表,用计量工具加料
- [ ] **检查方法**: 厨师长每班抽查不少于3道菜的调料加量,对比是否符合标准
- [ ] **工具配置**: 每个炉灶配备0.5g精度的厨用电子秤和标准量勺

**关键工具**
- [ ] 厨用电子秤 (精确到0.5g)
- [ ] 标准量勺 (5ml、10ml、15ml各一把)
- [ ] 调料配比标准表 (贴于炉灶旁)

#### A4-08 出菜秩序与传菜协调

**风险描述**
- 问题: 菜品出菜顺序混乱,导致顾客菜品到达时间不一致(有的菜冷了,有的还没到)
- 根因: 出菜和传菜没有明确的协调机制

**防呆执行标准**
- [ ] **出菜秩序**:
  1. 热菜厨师按照小票单据顺序出品 (不可跳单或乱序)
  2. 出完后贴在菜品旁,等待传菜员取菜
  3. 若同一桌有多道菜,先出的菜等后出的菜(齐菜出)
  - [ ] **传菜协调**:
  1. 传菜员监看热菜台,菜品出来后立刻取菜
  2. 若有多道菜,等齐了一起送,防止顾客单菜先到(冷了)
  3. 取菜时通报菜品名称和桌号,确认无误
- [ ] **延误处理**: 若超过3分钟菜品未被取走,热菜厨师主动询问传菜员"菜品何时取?"防止菜品放置过久(变冷)

**关键表单**
- [ ] 出菜顺序记录表 (记录出菜时间顺序)
- [ ] 传菜员工作清单 (记录取菜时间,防止遗漏)

#### A4-09 设备维护与应急预案

**风险描述**
- 问题: 炉灶、锅、温度计等关键设备损坏,导致出菜无法进行
- 根因: 平时维护不到位,故障时应急反应慢

**防呆执行标准**
- [ ] **日常维护检查**:
  - 开市前: 检查炉灶点火是否正常、锅是否无破损、温度计是否可用
  - 营业中: 每2小时检查一次设备运转状态
  - 收市后: 清洁设备,检查是否有故障迹象
- [ ] **维护记录**: 填写《设备维护日志》,记录检查结果和任何维修需求
- [ ] **应急预案**:
  - 若主炉灶故障: 立刻启用备用炉灶,暂停部分菜品,告知前厅
  - 若温度计故障: 立刻更换新温度计,已出品的菜品进行手动温度检查
  - 若小票机故障: 改用纸笔记单,按顺序烹饪,防止漏单
  - 若冰箱故障: 立刻通知厨师长和店长,评估食材是否可用,必要时启动应急采购
- [ ] **维修跟踪**: 故障报修后,必须跟踪维修进度,确保及时修复

**关键表单**
- [ ] 设备维护日志 (每班填写)
- [ ] 应急预案联系电话 (贴于炉灶旁)

### 2.5.5 进销存管理

#### 备货标准
- [ ] 根据环比、同比进行开货
- [ ] 根据货品到货周期核算出门店安全保有量
- [ ] 根据实际运营、周期、保存时限进行备货
- [ ] 根据每日营业情况预制各类小吃、包子等按备货量表的备货清单进行备货

#### 领货标准
- [ ] 干杂类辅料库房领货
- [ ] 食材类根据单日销售量与剩余量,所须第二天物料数量上报至厨师长每日下单
- [ ] 供货商早上送货
- [ ] 馅料/辅料类产品,库房领货并做好登记

#### 领货验收标准
- [ ] 苕粉干燥、无霉变现象
- [ ] 酥肉无粘连现象,呈金黄色,色泽均匀,无肉眼可见杂质
- [ ] 蔬菜不能有腐烂现象
- [ ] 面粉无霉变现象,无肉眼可见杂质
- [ ] 具有该产品固有的气味,无异味
- [ ] 称重
- [ ] 预包装食材外包装完好,无破损、污染、明显挤压、无漏气、渗漏等现象
- [ ] 玻璃器皿盖子无破碎且正确与容器密封
- [ ] 原料是否存在颜色、气味等异常或已变质现象
- [ ] 检查效期,产品未超过使用期限,且有充足的时间供使用

#### 加工标准

**现炸酥肉:**
- [ ] 打开包装取出酥肉160g
- [ ] 将电炸炉油温调至180℃的刻度,指示灯亮后下酥肉炸制
- [ ] 将酥肉炸呈金黄色捞出沥干油
- [ ] 装盘,酥肉旁边加5g干碟即可出品

**油炸类小吃制作通用标准:**
- [ ] 打开包装取出食材(冷冻食材注意是否需要解冻)
- [ ] 将电炸炉油温调至180℃(允许误差±10℃),使用油温计测量确认,或观察指示灯亮后下料炸制
- [ ] 炸制时间根据食材大小和类型决定,炸至表面金黄色即可捞出
- [ ] 沥干油分后装盘,添加配料(糖浆、调味粉等)按门店标准出品

**煎制类小吃制作通用标准:**
- [ ] 调制面浆:按照门店配方手册调制(粉类+水+调味料+蛋液等),比例准确
- [ ] 将原料放入容器中用打蛋器打散至无颗粒,过筛后备用
- [ ] 电饼档或煎锅调至180-200℃,预热3-5分钟至温度稳定,用刷子在锅底均匀刷一层油(约5ml)
- [ ] 按标准份量倒入面浆,煎制至表面干燥、色泽金黄
- [ ] 装盘后按需切块,添加配料(辣椒、葱花等调味料)后出品

**鱿鱼改刀:**

- [ ] 再以35°的斜刀切丝,深度3/4、宽度5cm一段的麦穗花
- [ ] 分切后的鱿鱼放入保鲜盒中,加入开水将鱿鱼烫定型后换清水浸泡备用

**内脏类涮品改刀通用标准:**
- [ ] 冷冻食材需提前软化处理(软化时间根据大小决定,一般10-20分钟)

- [ ] 改刀时注意保持食材完整性,深浅一致,便于涮食和入味

**肉类涮品切片通用标准:**
- [ ] 去除表面筋膜及不可食用部位
- [ ] 需要腌制去腥的食材:浸泡处理(时间根据食材而定,一般20-40分钟),腌制液温度保持低温
- [ ] 切片标准:厚薄均匀,大小适中,便于涮食

### 2.7.5 加工标准

**辅料加工通用标准:**

**菌菇类辅料:**

- [ ] 葱类:切成3-5cm长的段(大葱切5cm、小葱切3cm),便于提味和夹取
- [ ] 姜类:洗净后切片(厚度2-3mm)或切块(1cm见方)
- [ ] 其他辅料:根据用途切配
  - 锅底用:切大块(蒜瓣对半切、香菜切5cm段)
  - 调味用:切碎末(蒜末、姜末直径≤3mm)
  - 装饰用:切丝或切花(葱丝长5cm宽2mm、香菜叶保持完整)

**鸭血制作:**
- [ ] 将1000ml饮用水+10g盐搅拌均匀
- [ ] 加入100ml乳制品搅拌均匀
- [ ] 加入1袋解冻好的生鸭血(约500ml)充分搅拌均匀
- [ ] 分装至直径15cm的碗中,每碗装7-8分满(约300ml),分装时要不停搅拌保持均匀,静置20分钟凝固
- [ ] 用尖刀均匀划开备用、按标准份量出品

**炸蛋制作:**
- [ ] 锅中倒入色拉油、油温控制在140℃
- [ ] 油温到140℃后蛋液用4两勺装满一勺倒入20cm的密漏均匀的漏入锅中
- [ ] 把锅中的蛋平整固定在20cm的密漏中炸至两面金黄、捞出控油备用

# 第三部分:设施设备管理

## 3.1 刀具、菜墩的保管及使用

**刀具的使用规范:**
- [ ] 刀具清洁完后擦干水渍,放置于指定位置挂好或存放于消毒柜中
- [ ] 刀具不使用时放在固定位置,避免掉落伤人
- [ ] 各类刀具按照规定的要求使用,防止交叉污染
- [ ] 保证每天清洗消毒,避免刀具生锈
- [ ] 保持刀具的卫生、干净、整洁

**菜墩的使用规范:**
- [ ] 菜墩时刻保持干净、干燥,立体摆放
- [ ] 菜墩分开存放并做标识标贴
- [ ] 使用后及时清理残渣并清洗干净
- [ ] 保证每天清洗消毒,避免发霉
- [ ] 菜墩进行分类使用,避免交叉污染

**刀具消毒:**
- [ ] **化学消毒浸泡法(84消毒液)**:
  - 比例: 1:200 (即1份消毒液+199份水)
  - 具体操作: 10ml消毒液 + 1990ml水 = 2000ml溶液
  - 或简便方法: 1瓶盖(约5ml)消毒液 + 1瓶(1000ml)矿泉水
  - 有效氯浓度: 约250mg/L
  - 浸泡时间: 30分钟
  - 浸泡后用清水冲洗干净
- [ ] **刀具消毒柜消毒**:采用紫外线、远红外线、高温、臭氧等方式进行消毒,消毒时长45-60分钟

**菜墩消毒:**
- [ ] **洗烫消毒**:先用硬刷和清水将菜板表面的缝隙洗刷干净,再用100℃以上的开水冲洗一遍
- [ ] **撒盐消毒**:每次用完菜墩后,用刀将板面残渣刮净,每隔6-7天在板面上撒一层盐,既可杀菌亦可防止菜墩干裂
- [ ] **塑料菜板化学消毒(84消毒液)**:
  - 比例: 1:200 (即1份消毒液+199份水)
  - 具体操作: 10ml消毒液 + 1990ml水 = 2000ml溶液
  - 或简便方法: 1瓶盖(约5ml)消毒液 + 1瓶(1000ml)矿泉水
  - 有效氯浓度: 约250mg/L
  - 浸泡时间: 30分钟
  - 浸泡后用清水冲洗干净

**注意事项**
- [ ] 84消毒液与洗洁精不可混用(会产生有毒氯气)
- [ ] 消毒后必须用清水冲洗干净,避免残留
- [ ] 配制好的消毒液当天使用,不过夜(有效氯会挥发)
- [ ] 操作时必须佩戴手套,避免皮肤直接接触
- [ ] 保持通风,避免吸入刺激性气体

## 3.2 燃气炒炉的使用方法及注意事项

**管理要求:**
- [ ] 灶台干净无杂物,工作中保持灶台卫生
- [ ] 完成一个菜品时须将火关到最小

**检查标准:**
- [ ] 水/电/气正常;无泄露,台面干净整洁

**使用标准:**
1. [ ] 接通电源
2. [ ] 打开燃气阀
3. [ ] 打开长明火
4. [ ] 点燃长明火
5. [ ] 调节风力大小
6. [ ] 调节火力大小
7. [ ] 使用完后关闭燃气阀
8. [ ] 关闭电源
9. [ ] 清理炉台卫生

**注意事项:**
- [ ] 炒炉运作过程中,必须开启室内排气装置,保持室内空气流通顺畅
- [ ] 火种小火未点燃或不正常时,切勿打开主火开关阀
- [ ] 中途熄火需要重新点炉时,关上所有气阀,开启风机吹掉聚积气体,然后再按操作规程重新操作
- [ ] 短时间暂停使用,可仅关闭主气阀,保持火种燃烧
- [ ] 熄炉时,应先关闭燃气,然后关闭风机。防止炒炉在燃烧状态下关闭风机,这样会产生黑烟
- [ ] 不得擅自改装炒炉,减少或增加装置、零部件等
- [ ] 遇到突发事故,应先切断燃气,关闭风机,上报处理

## 3.3 蒸饭车的使用方法及注意事项

**管理要求:**
- [ ] 燃气蒸饭车使用由厨师长指定人员负责操作,对新上岗人员应提前两天培训操作规程
- [ ] 在准备工作完成后,点火前应先检查前灶堂内水位情况,无水不得点火使用
- [ ] 设备内外干净整洁

**检查标准:**
- [ ] 设备无故障、蒸饭车表面及内部无残渣、水垢、油渍

**使用标准:**
1. [ ] 打开燃气阀
2. [ ] 打开柜门,将已装好米或面食的饭盘放进柜内,关好并锁紧柜门
3. [ ] 使用时打开电子打火阀,往顺时针方向转动旋钮,听到咔叭的打火声
4. [ ] 从观火孔观察柜内燃烧器是否点燃,火焰稳定燃烧10-15秒后松开旋钮(过早松开会熄火)
5. [ ] 箱门保持关闭
6. [ ] 蒸箱火力开关(可调大小)

**注意事项:**
- [ ] 使用前先检查气阀开关是否开启
- [ ] 蒸箱每日换水,使用时,须将水加到规定位置
- [ ] 不准在蒸饭车使用过程中打开蒸饭车查看食品生熟情况或搅拌汤锅,以免蒸气灼伤
- [ ] 打开蒸柜门时,蒸汽会散去,注意不要碰触到蒸汽及蒸柜四周,以免烫伤
- [ ] 为了避免蒸柜的供水系统因水垢过多引起堵塞,使用前必须检查

## 3.4 刨肉机使用方法及注意事项

**管理要求:**
- [ ] 使用前必须检查刨肉机工作是否正常
- [ ] 使用时,必须按正确流程操作
- [ ] 使用后,必须清理干净,保证无杂物

**检查标准:**
- [ ] 检查刀盘和轴承座,设备正常,无故障

**使用标准:**
1. [ ] 接通电源
2. [ ] 拆开压杆
3. [ ] 放入食材类
4. [ ] 调整刀片刻度
5. [ ] 按启动键开始工作
6. [ ] 使用完后关闭自动键
7. [ ] 关闭电源
8. [ ] 日常卫生清理

**注意事项:**
- [ ] 操作员严禁带任何手饰、手链操作此设备,防止发生钩挂事故
- [ ] 设备开启前,检查设备电源、开关是否正常、漏电
- [ ] 使用前调整好刀片所需要的厚薄度,固定好所刨物品,然后开机;严禁手接触刀片
- [ ] 刨肉时,操作员不得离开,做到人离停机
- [ ] 禁止切结冰的食物、连骨的肉类、鱼类及非食物类任何物品
- [ ] 操作完成后,关闭电源、清洁设备,严禁使用水冲洗
- [ ] 发生异常情况,立即关闭电源,停止使用,并通知相关人员进行检查和维修
- [ ] 换肉程序:先关机、取肉、夹肉、开机刨肉
- [ ] 操作刨肉机时为安全起见严禁佩戴手套,但必须佩戴袖套;操作完成处理食材时立即佩戴一次性手套

## 3.5 冻库的使用与维护

**管理要求:**
- [ ] 冷冻库只存放厨房备用食品、原料及其盛器,不得存放其他杂物;职工私人物品一律不得存入其内
- [ ] 坚持冻藏食品及原料必须处在冰冻状态下才进入冻库的原则,避免将已经解冻的食品及原料送入冻库
- [ ] 所有冻藏食品及原料必须注明入库日期;区别库存食品及原料不同种类、性质、固定位置,分类存放,并严格遵守冻藏期限的规定
- [ ] 加强对冻藏品计划管理,坚持"先存放,先取用"的原则,交替存货和取用
- [ ] 冻藏大块原料单独存放,小型及零碎原料置盘、筐内集中存放;所有物品必须放在货架上,并至少离地25cm以上,离墙5cm以上
- [ ] 定期对冷冻库进行清洁整理,定期检查食品及原料质量,并定期对冷冻库进行清理、消毒、预防和杜绝鼠害发生,保持其卫生整洁
- [ ] 控制有权进入冷冻库的人员数量,计划、集中领货、减少库门开启次数;由专人每周定期盘点库存情况,报告厨师长
- [ ] 经常检查,保持冷冻库达到规定的温度;如发现温度偏差,应及时报告厨师长与工程部联系解决,不得自行修理

**检查标准:**
- [ ] 物品按4D标准摆放,做好标识标贴,卫生干净整洁;抽查食材效期

**使用标准:**
- [ ] 冻库一般分为两部分:保鲜室和冷冻室
- [ ] 保鲜室主要存放新鲜蔬菜及需冷藏物品
- [ ] 冻库主要存放需冷冻的食材
- [ ] 物品摆放按照4D标准,分类储存,拿取产品必须做到先进先出
- [ ] 按照周清计划表对冻库进行卫生打扫及除冰
- [ ] 保鲜室温度:±5℃;冷冻室控制在-18℃(±2℃)

**注意事项及故障处理:**
- [ ] **温度异常处理**:
  - 保鲜室温度>7℃或<3℃,立即调整温控旋钮,30分钟后复查
  - 冷冻室温度>-16℃或<-20℃,立即调整温控旋钮,1小时后复查
  - 调整后仍不达标,立即报修并将食材转移至备用冷柜
- [ ] **设备系统故障处理**:
  - 通风系统异常(风扇不转、异响):立即关停设备,报修,转移食材
  - 电力系统异常(跳闸、漏电):立即断电,报修,不得私自维修
  - 管道系统异常(漏水、结冰):立即关停设备,清理积水,报修
- [ ] **故障等级评估**:
  - 轻微故障(温度偏差<2℃、轻微异响):可继续使用,当日报修
  - 中度故障(温度偏差2-5℃、明显异响):转移部分食材,2小时内报修
  - 严重故障(完全不制冷、漏电):立即关停,转移全部食材,紧急报修

**检查项不符处理流程**:
1. [ ] **发现不符**:立即记录在案,不打勾
2. [ ] **原因分析**:分析是设备故障、流程问题还是人为问题
3. [ ] **即时改善**:现场立即采取措施纠正
4. [ ] **效果验证**:改善后重新检查,验证是否达标
5. [ ] **记录追溯**:记录问题、处理方式、验证结果,便于后续改进

**责任人签字说明**:
- **检查人员签字**:表示该项已亲自检查并确认
- **责任岗位签字**:表示该岗位接受检查结果,承诺改进
- **厨师长签字**:表示已复核,确认符合标准
- **缺少任何签字,本清单作废,需重新检查**

**常见问题Q&A**:

**Q1:温度计损坏了,怎么办?**
A: 立即上报厨师长,用备用温度计替代,修好后写在"备注"栏,不影响检查完成

A: 填写具体整改内容和完成时间,整改完成后再补打勾,必须厨师长确认

A: 立即关闭,记录"发现时间",标注在检查清单上,不能隐瞒

A: 不能。必须由实际执行人签字。厨师长只能在"复核签字"栏签字

# 第四部分:应急管理

> **说明**: 厨房常见突发情况的应急处理流程,确保快速响应

## 4.1 设备故障应急预案

### 4.1.1 燃气灶故障
1. **立即停用**:张贴"故障停用"标识,通知厨师长
2. **评估影响**:厨师长判断能否用其他灶台替代
3. **调整出餐**:无法替代时,通知前厅暂停相关菜品
4. **联系维修**:记录故障时间和原因,联系维修人员
5. **恢复营业**:维修完成后测试正常,恢复菜品供应

### 4.1.2 冰箱故障
1. **食材转移**:
   - 有备用冰柜:立即将食材转移至备用冰柜
   - 无备用冰柜:紧急采购冰块临时降温,或联系附近门店/供应商协助冷藏
2. **温度检查**:检查食材温度,超过8°C的易腐食材做报废处理
3. **损失记录**:报废食材拍照留证,填写《设备故障报损表》
4. **联系维修**:通知维修,预计修复时间
5. **调整菜单**:维修期间暂停受影响的菜品供应
6. **后续处理**:总结原因,避免再次发生

### 4.1.3 炸缸/刨片机等专用设备故障
- 参照上述流程:停用→评估→调整→维修→恢复


## 4.2 能源中断应急预案

### 4.2.1 停电
1. **暂停出餐**:立即通知前厅暂停需要电力的菜品
2. **应急措施**:启用应急照明,冰箱避免频繁开启
3. **联系沟通**:联系物业/电力公司,询问恢复时间
4. **调整方案**:超过30分钟,考虑仅供应部分菜品或暂停营业
5. **恢复检查**:来电后检查设备是否正常,确认安全后恢复营业

### 4.2.2 燃气中断
1. **暂停热菜**:立即通知前厅暂停需要燃气烹饪的菜品
2. **检查原因**:确认是燃气余额不足还是供应中断
3. **应急补充**:燃气余额不足时,立即联系充值
4. **调整菜单**:供应中断时,临时仅供应凉菜和小吃


## 4.3 食品安全应急预案

### 4.3.1 发现食材异常
1. **立即停用**:可疑食材立即停用,单独存放
2. **排查范围**:检查同批次食材是否都有问题
3. **通知厨师长**:汇报情况,厨师长决定是否全部报废
4. **联系供应商**:拍照留证,联系供应商退换货
5. **记录存档**:填写《食材异常记录表》,追溯原因

### 4.3.2 顾客食品安全投诉
1. **保留样品**:立即保留该菜品样品和剩余食材
2. **配合调查**:配合店长/厨师长调查原因
3. **详细记录**:记录菜品制作过程、食材来源、操作人员
4. **后续改进**:根据调查结果,改进流程,避免再次发生
5. **参照预案**:详细处理参照《食品安全应急预案》(如有)


## 4.4 人员突发情况应急预案

### 4.4.1 员工突然缺勤(生病、请假等)
1. **临时调配**:厨师长立即调配其他岗位人员填补
2. **简化菜单**:必要时与店长沟通,临时减少部分菜品供应
3. **加班协调**:协调其他员工加班,确保出餐效率
4. **长期规划**:频繁缺勤的岗位,考虑储备替补人员

### 4.4.2 高峰期人手不足
1. **厨师长补位**:厨师长填补最紧急的岗位
2. **优先处理**:优先处理堂食订单,外卖适当延后
3. **通知前厅**:告知前厅出餐时间可能延长
4. **事后总结**:班前会总结经验,优化备餐流程

## 5.1 培训周期与阶段

### 培训周期
**标准培训期:10-14天(根据个人能力弹性调整)**
- 快速学习型:10天即可独立上岗
- 普通学习型:12-14天独立上岗
- 特殊情况:延长至14天以上,不轻易放弃新人

### 培训阶段

#### 第1-3天:观摩学习
**目标**:熟悉厨房环境和基本流程
- [ ] 师傅演示操作,新人观察记录
- [ ] 鼓励新人多问问题,不要有压力
- [ ] 了解岗位职责和工作流程
- [ ] 熟悉设备使用和安全规范
- [ ] 无考核压力,重点是建立信心

#### 第4-7天:辅助操作
**目标**:参与简单任务,逐步上手
- [ ] 新人协助师傅完成工作
- [ ] 从简单任务开始(食材整理、备餐准备等)
- [ ] 逐步承担更多责任
- [ ] 师傅随时指导和纠错
- [ ] 允许犯错,重点是学习改进

#### 第8-10天:独立操作(师傅在旁)
**目标**:独立完成工作,师傅观察指导
- [ ] 新人独立完成岗位工作
- [ ] 师傅在旁观察,不主动干预
- [ ] 只在出现问题或偏差时才指导
- [ ] 逐渐减少师傅的帮助
- [ ] 培养新人的自主判断能力

#### 第11-14天:独立上岗
**目标**:完全独立工作,偶尔抽查
- [ ] 新人完全独立完成岗位工作
- [ ] 师傅偶尔抽查,不再全程监督
- [ ] 厨师长进行最终评估
- [ ] 达标后正式转正


## 5.2 考核标准

### 实操考核(不搞笔试,以实际操作为主)
- [ ] 能独立完成岗位日常工作
- [ ] 出品质量符合标准(品相、口味、温度)
- [ ] 遵守安全和卫生规范
- [ ] 能按时完成备餐和出餐任务

### 弹性评估原则
- 不设固定天数,能力达标即可提前转正
- 不设固定分数线,综合评估实际表现
- 考核不合格不直接劝退,延长培训期或调岗
- 给予新人足够的成长时间和改进机会


## 5.3 师傅激励机制

### 培训期奖励
- **新人14天内独立上岗**:师傅奖励300元
- **新人转正后3个月无重大问题**:师傅再奖励200元

### 带教支持
- [ ] 师傅需耐心指导,不得嘲笑或批评新人
- [ ] 每日向厨师长汇报新人进展情况
- [ ] 发现问题及时沟通,共同解决
- [ ] **工作量调整**:新人培训期间(前7天),师傅工作量适当减轻
  - 厨师长协调其他岗位分担部分工作
  - 或临时减少该档口部分菜品供应
  - 高峰期厨师长协助该档口出餐

### D.1 食材处理示例

#### 示例1:鲜牛肉类涮品处理(以某门店为例)

**鲜牛肉切片案例:**
- 选择新鲜牛肉(后腿肉、里脊肉等),检查色泽鲜红、无异味
- 顺着肉纹理方向切成大块,便于后续逆纹切片
- 将牛肉冷藏至半冷冻状态(约-2℃至0℃),便于切片
- 逆着肉纹理方向切薄片,厚度约2-3mm,确保厚薄均匀
- 切好的牛肉片立即放入0-4℃冷藏保存,当餐使用

> **参考要点**: 鲜牛肉切片的关键是"逆纹切",可使肉质更嫩;半冷冻状态便于切片;厚度2-3mm适合涮煮;全程保持低温(0-4℃)保证新鲜度。

---

**牛肉丸制作案例:**
- 选用新鲜牛肉500g(去筋膜),切小块
- 加入冰水50ml、盐5g、白胡椒粉2g,放入绞肉机搅打至起胶(约5-8分钟)
- 将肉泥冷藏30分钟,使其更有弹性
- 手工捏成丸子状(每个约15-20g),表面光滑无裂纹
- 放入80-90℃热水中定型3-5分钟,捞出过凉水
- 冷藏保存(0-4℃),当日制作当日使用

> **参考要点**: 牛肉丸需要"打至起胶"才有弹性;加入冰水可保持低温,防止出油;定型温度80-90℃不宜过高;当日制作当日使用保证口感。

---

**嫩牛肉腌制案例:**
- 选用牛里脊或嫩牛肉500g,切片(厚度约3mm)
- 腌制配方:生抽10ml、料酒10ml、蛋清1个、淀粉5g、食用油5ml
- 将所有腌制料加入牛肉片中,轻柔抓匀(避免过度揉搓破坏肉质)
- 冷藏腌制20-30分钟,使其入味
- 腌制好的牛肉需当日使用完毕

> **参考要点**: 嫩牛肉腌制加入蛋清和淀粉可"锁水保嫩";腌制时间20-30分钟适中;轻柔抓匀避免破坏纤维;冷藏保存保证新鲜。

---

#### 示例2:根茎类蔬菜处理(以某门店为例)

**小土豆处理案例:**
- 挑选:选择大小均匀,无发芽无青皮
- 清洗:流水冲洗3遍,刷洗表面
- 保留外皮(营养价值高)
- 整颗使用,无需切割
- 蒸制:蒸至筷子能轻松插入即可

> **参考要点**: 根茎类食材可保留外皮增加营养,清洗要彻底去除泥沙,蒸制时间以能轻松插入为准。

---

**小芋头处理案例:**
- 去皮:削去外皮
- 切块:切小块,便于蒸制和食用
- 浸泡:浸泡清水防止氧化变色
- 蒸制:蒸至软糯

> **参考要点**: 易氧化的根茎类需要浸泡,切块大小影响蒸制时间。

---

#### 示例3:叶菜类蔬菜处理(以某门店为例)

**茼蒿处理案例:**
- 挑选:摘除黄叶、老梗
- 清洗:浸泡10分钟+流水冲洗3遍
- 沥水:使用沥水篮,沥水5分钟以上
- 切配:保留10-15cm长度,去除根部

> **参考要点**: 叶菜类需要浸泡去除农药残留,沥水要充分避免稀释汤底。

---

**油麦菜处理案例:**
- 摘除老叶、黄叶
- 浸泡清洗,流水冲洗
- 沥干水分
- 切配:保留10-15cm长度,去除根部,便于夹取

> **参考要点**: 处理流程与茼蒿类似,关键是挑选、浸泡、沥水。

---

#### 示例4:菌菇类食材处理(以某门店为例)

**香菇处理案例:**
- 去根:去除根部较硬部分
- 清洁:用湿布擦拭表面(避免水洗流失营养)
- 改刀:表面改十字花刀,利于入味
- 储存:避光保存,当日使用

> **参考要点**: 菌菇类优先用湿布擦拭而非水洗,花刀有助于入味。

---

**平菇处理案例:**
- 撕成小块状
- 去除根部
- 检查无杂质
- 当日使用

> **参考要点**: 平菇等大型菌菇可撕成小块,便于烹饪。

---

#### 示例5:豆制品处理(以某门店为例)

**豆皮处理案例:**
- 检查保质期和新鲜度
- 流水冲洗干净
- 切宽条,约3cm宽
- 开封后4小时内使用完毕

> **参考要点**: 新鲜豆制品保质期短,开封后要尽快使用。

---

**冻豆腐处理案例:**
- 自然解冻或冷藏解冻
- 轻轻挤压去除多余水分
- 切块,大小适中
- 未用完重新冷冻并标注日期

> **参考要点**: 冷冻豆制品需要正确解冻,挤压水分影响吸收汤汁的能力。

### D.2 小吃制作示例

#### 示例1:野菜卷制作(以某门店为例)

**原料准备:**
- 春卷皮10张(或馄饨皮)
- 野菜馅料300g(野菜剁碎、加入调味料拌匀)
- 鸡蛋1个(打散,用于封口)

**制作流程:**
- 将春卷皮平铺在操作台上,中间放入约30g野菜馅料
- 从一端开始卷起,卷至一半时将两边向内折叠
- 继续卷至末端,用蛋液封口,确保封口紧密不漏馅
- 将电炸炉油温调至170-180℃,指示灯亮后下野菜卷炸制
- 炸制约2-3分钟,表面呈金黄色、外皮酥脆时捞出沥油
- 装盘,可配干碟或蘸料

> **参考要点**: 野菜卷的关键是馅料不能太湿(否则易破皮),封口要严实(防止炸制时漏馅);油温170-180℃适中,炸至金黄酥脆;沥油充分保证口感。

---

#### 示例2:空气丸子制作(以某门店为例)

**调浆配方案例:**
- 淀粉类(红薯淀粉或土豆淀粉)200g
- 水300ml
- 鸡蛋2个
- 食盐3g
- 泡打粉2g(可选,增加蓬松度)

**制作流程:**
- 将所有原料放入容器中用打蛋器充分搅拌至无颗粒,静置5分钟
- 搅拌好的面糊过筛,去除未打散的颗粒和气泡
- 将电炸炉油温调至170℃,用小勺舀起面糊(约15-20g)
- 将面糊慢慢滑入油中,炸制过程中丸子会自然膨胀变大
- 炸制约1.5-2分钟,丸子膨胀至原来2-3倍大小,表面金黄即可捞出
- 沥干油后装盘,可撒葱花或配蘸料

> **参考要点**: 面糊的稀稠度很重要(水淀粉比约1.5:1),过稠不易膨胀,过稀易散;油温170℃适中(过高外焦里生,过低吸油多);丸子会自然膨胀成"空气丸子",内部中空;炸制时间不宜过长。

---

#### 示例3:怪噜洋芋制作(以某门店为例)

**原料准备:**
- 土豆500g(中等大小)
- 淀粉50g
- 食用油(炸制用)

**制作流程:**
- 土豆去皮,切成滚刀块(大小约3-4cm)
- 将土豆块放入蒸锅,蒸至7-8成熟(筷子能插入但仍有硬心)
- 取出土豆块,趁热裹上一层薄薄的干淀粉
- 将电炸炉油温调至180℃,下入裹好淀粉的土豆块
- 炸制约3-4分钟,表面形成金黄色硬壳,内部软糯时捞出沥油
- 装盘,撒上特制调料粉(辣椒粉、孜然粉、椒盐等混合)或配蘸料

> **参考要点**: 土豆蒸至7-8成熟是关键(全熟易碎,太生内部不够软);裹淀粉要均匀且薄(太厚口感不好);油温180℃,炸至外壳硬脆、内部软糯形成"怪噜"口感;趁热撒调料粉更易附着。

### D.3 辅料加工示例

#### 示例1:锅底辅料加工(以某门店为例)

**番茄处理:**
- 挑选新鲜番茄,色泽鲜红、无损伤
- 清洗干净,去除蒂部
- 切成圆形厚片,厚度约1-1.5cm
- 或切成4-6瓣(根据番茄大小)

**木姜子处理:**
- 挑选颗粒饱满、无霉变的木姜子
- 用清水冲洗干净
- 整颗使用或轻轻拍碎(使其更易释放香味)
- 用量适中,避免香味过重

**葱段处理:**
- 选择新鲜大葱,葱白部分为主
- 清洗干净,去除老叶和根须
- 切成约5-8cm长段
- 或拍松后切段,更易释放香味

**香料类辅料处理:**
- 根据配方选择八角、桂皮、香叶等
- 检查无霉变、无异味
- 干香料直接使用,无需清洗(避免流失香味)
- 用纱布包或料盒装,便于取出

> **参考要点**: 锅底辅料的切配要考虑其在锅底中的作用:番茄释放酸味和红色,切片或切瓣均可;木姜子提供特殊香味,拍碎更易释放;葱段增香,拍松效果更好;干香料用纱布包装便于取出,保持汤底清爽。

---

### D.4 海鲜类加工示例

#### 示例1:鱿鱼改刀(以某门店为例)

**制作流程:**
- 将鱿鱼抽出主筋,从中切断,撕去背部筋膜,清洗干净
- 以45°的斜刀切片,深度3/4、宽度1cm
- 再以35°的斜刀切丝,深度3/4、宽度5cm一段的麦穗花
- 分切后的鱿鱼放入保鲜盒中,加入开水将鱿鱼烫定型后换清水浸泡备用

> **参考要点**: 鱿鱼改麦穗花需要两次斜刀,角度和深度都要控制好(约3/4深度,不切断),开水烫定型可使花纹更明显。

---

#### 示例2:鲍鱼改刀(以某门店为例)

**制作流程:**
- 选择新鲜鲍鱼或冷冻鲍鱼(冷冻需提前解冻)
- 用小刀沿着鲍鱼壳边缘将鲍鱼肉取出
- 去除内脏部分(黑色或深绿色部分),保留裙边
- 用软毛刷或牙刷轻轻刷洗表面黏液,清水冲洗干净
- 根据大小决定改刀方式:
  - 小鲍鱼(5cm以下):在表面打十字花刀,深度约1/2,便于入味
  - 大鲍鱼(5cm以上):切成0.3-0.5cm厚片,或打交叉花刀后切片
- 改好刀的鲍鱼放入冰水中浸泡(0-4℃),保持新鲜度

> **参考要点**: 鲍鱼要彻底去除内脏和黏液,否则有腥味;花刀深度约1/2,太深易碎,太浅不入味;大鲍鱼切片厚度0.3-0.5cm适合涮煮;全程保持低温(0-4℃)。

---

#### 示例3:鲈鱼改刀(以某门店为例)

**制作流程:**
- 选择新鲜鲈鱼(眼睛明亮、鳃红色、鱼鳞完整)
- 去鳞:用刀背或去鳞器从尾部向头部刮除鱼鳞
- 开膛:从腹部剖开,取出内脏,清洗干净腹腔内的血水和黑膜
- 去鳃:将鱼鳃完全剔除(鱼鳃是腥味主要来源)
- 改刀方式(根据出品形式选择):
  - **鱼片**:从背部下刀,沿着鱼骨片下鱼肉,去除鱼刺,切成约0.3-0.5cm厚片
  - **鱼块**:将鱼身切成约2-3cm宽的段,保留骨头(更鲜美)
  - **花刀**:在鱼身两侧每隔1-2cm斜切一刀,深至鱼骨(便于入味和定型)
- 改好刀的鱼肉放入冰水中漂洗2-3遍,去除血水和黏液
- 沥干水分后放入0-4℃冷藏,当日使用

> **参考要点**: 鲈鱼必须去除鱼鳃和腹腔黑膜,否则极腥;鱼片厚度0.3-0.5cm适合涮煮,过厚不易熟;冰水漂洗可去腥并使肉质更紧实;鱼骨和鱼头可用于熬汤;当日加工当日使用保证新鲜。
